💥We can't hide it anymore so here it goes... **Test Action is BACK!** 💥
And with it, a list of fixes, improvements, and some new actions to play with.

### **New Features**

### Test Action

- You can now test almost all actions before using them in your processes.&#x20;

![](/images/docs/ZlM1ZF1_4sNceOpsOdVMk_generated-guid-test.gif)

- Just define some variables, fill in the values and hit **Test action**.

![](/images/docs/c0NdnuE-8h4rgKBXvnyW5_add-test.gif)

- Make sure to check the **Test Status&#x20;**&#x73;ection and keep an eye out for errors. 😉

![](/images/docs/QVshep5eAcCHbTguwd0vb_divide-test.gif)

- Testing actions that work with files or lists of files is also possible.

![](/images/docs/JtFjbvrdtaonsDyuO_XT1_zip-test.gif)

- You can even test **Call API.&#x20;**

![](/images/docs/Ptes89WdgZcZsr5ooPyBn_call-api-test.gif)

- Or that anoying **JPath&#x20;**&#x63;onfiguration from **Json Mapper**.

> \&#123;
> &#x20;   "firstName": "John",
> &#x20;   "lastName": "Doe",
> &#x20;   "age": 26,
> &#x20;   "address": \&#123;
> &#x20;       "streetAddress": "Main Street",
> &#x20;       "city": "New York",
> &#x20;       "postalCode": "6300192"
> &#x20;   &#125;
> &#125;

![](/images/docs/SkB_EsKOTeFN7o6HeRGRO_json-mapper-test.gif)

- However, **Test Action** is not available for the following actions (*at least* *not yet* 😎)
  - **Decisional**
  - **Start**
  - **Stop**
  - **Delay**
  - **For Each**
  - **Throw**
  - **Generate Document**
  - **Call Subprocess**
  - **Join**
  - **Map Process Data**

### OAuth 2

- If you've read all the way down here, we want to let you know that **OAuth Stage I** is also live, you can set up OAuth 2 with your third-party applications. Yes, some configurations will be needed, until we'll bring pre-configured apps with OAuth 2.&#x20;
  This will be part of **OAuth Stage II**. We're on it, so stay tuned! 📻

### **Improvements**

- When using **Add Variable**, the datatype and **IsList&#x20;**&#x66;lag are now selected by default.

![](/images/docs/7GkNbZ8shVBwawHtJqram_autoselect-datatype.gif)

- You can now see a breakdown of execution time in the **Instances View**.

![](/images/docs/1hK4gtxJKWlF68miOSIIr_analytics-instance.gif)

- You can also see if your instance was **triggered** by a schedule or some user.

![](/images/docs/IGaqihW6iXAQWvV10gS39_who-launched-instance.gif)

**API key -&#x20;**&#x77;e now support up to 25 API keys per user per workspace

![](/images/docs/KJOFTJy03xIHEw3jm2Q0G_image.png)
