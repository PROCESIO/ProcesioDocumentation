Importing and exporting resources can be a useful tool when you want to reuse work from one workspace in another workspace. With PROCESIO, you can export a large number of resources, like processes, credentials, data models, etc., and import them into a new workspace.



<Callout type="warn">
Import is not available in Personal workspaces.
</Callout>

<Callout type="info">
You must have an Admin role to access Export/Import functionality in a workspace.
</Callout>


Here's a simple step-by-step guide on how to import and export resources in PROCESIO:

First, open your PROCESIO workspace and click on the "Import/Export" button.&#x20;

![](/images/docs/elZYnKJofKLE9wfAjQOkv_group-673.png)

## Export

1. To export resources, select **Export&#x20;**&#x61;ction.

![](/images/docs/GaujJoMA6jFw5vWgADGx3_screenshot-2023-04-19-at-132720.png)

2\. Give a name to your export file. Else, it will have a default name, i.e. `export.procesio`

3\. Next, select the type of resources you want to export.&#x20;

<Callout type="info">
You can select to export all resources by choosing **Export all**.
</Callout>

![](/images/docs/PVN93kfW7eN5sdt7mHbkH_screenshot-2023-04-19-at-133257.png)

4\. Click the **Export** button and a file with a "*.procesio*" extension will automatically start to download.

5\. Press **Close** to close the panel.



## Import

1. To import resources, select **Import&#x20;**&#x61;ction.

![](/images/docs/GaujJoMA6jFw5vWgADGx3_screenshot-2023-04-19-at-132720.png)

2\. To import resources, you'll need to upload a file with the "*.procesio*" extension by clicking on the "Upload file" button.

3\. Once the file is uploaded, you can set the import rules:

![](/images/docs/WT8ZL4lWhjJMtIcXtASLT_screenshot-2023-04-19-at-154957.png)

- override data - will override resources with the same Id (if resources have the same name, but different Id, will not override)
- do not override data - will skip resources with the same Id

See below an example of 2 resources having the same name but different IDs:

![](/images/docs/2HpNM4Als0Ewqv5viPnIy_group-673.png)

![](/images/docs/6-k702xnhoCJLKGHPXPRF_screenshot-2023-04-19-at-153359.png)

4\. You can choose to import all resources included in the file or select only certain resources to import.

5\. Click on the **Import** button, and the resources will start to import. Once the import is complete, a confirmation message will pop up saying "Import completed."

That's it! You've successfully imported or exported resources in PROCESIO. This tool can be especially helpful when you want to transfer resources from one workspace to another, allowing you to quickly replicate processes and data models, and other types of resources without having to recreate them from scratch.
