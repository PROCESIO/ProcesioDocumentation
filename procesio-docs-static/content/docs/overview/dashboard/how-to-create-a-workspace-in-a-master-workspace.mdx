A master workspace is created for **administration purposes** of an enterprise account.&#x20;

Here you can create, delete or update:

- Workspaces
- Users

Use the dashboard to see in one place the usage consumption at the master workspace level in "Plan usage", and also at the workspace level in "Workspace usage":

![](/images/docs/5AzvzfmD1qQByu794JKO4_screenshot-2023-03-23-at-161651.png)

To create a new workspace in your master, you can:

1. Hit the "Create workspace" button in the Dashboard&#x20;

![](/images/docs/414TegCDfskHusGxOc9Xf_screenshot-2023-03-23-at-165028.png)

&#x20;       2\. Go to the "Workspaces" page using the left-side menu.

Press "Create workspace". Give it a name and set the process execution setting for the new workspace. Hit "Create" and the new workspace is created and added to the workspaces list.

![](/images/docs/4DlGo_6yXgYPDMlYV8jVH_ezcom-maker-3.gif)

Click the open in new tab icon to access the workspace:

![](/images/docs/txDIJOm9ml1Fxluv32xhe_screenshot-2023-03-23-at-162808.png)

Now you have successfully accessed the workspace you created. Here you can create processes automation, and integrations to solve all your business needs! 🚀
