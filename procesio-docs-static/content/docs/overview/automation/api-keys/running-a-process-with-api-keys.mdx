### Prerequisites

To run the process using a API Key we will require:

- Generate a API Key in PROCESIO.
- PROCESIO API endpoints can be found here. Additionally we also provide [Swagger for our Web API](https://webapi.procesio.app/swagger/index.html) endpoints which can be tested here.
- [Postman](https://www.postman.com/downloads/) for sending the request.

### Running a process from PROCESIO using API keys

**&#x20; Step 1.   &#x20;**&#x41;fter generating the PROCESIO API key, you need to copy and save the key name & key value. Key value will be hashed after navigating off the page.

**&#x20; Step 2.   &#x20;**&#x46;rom the PROCESIO API documentation, we will be looking for the POST method for running a process.

![](/images/docs/Cuswz2Iu9N_ylGrpJfnu6_swagger.png)

As observed in Swagger, the endpoint **/api/Projects/\&#123;id&#125;/run** requires three parameters:

- **Process ID** (the process ID, needs to be mentioned in the URL)
- **Workspace&#x20;**(needs to be mentioned in Headers)
- **Request body** (JSON, you are required to use the one provided below)

```json
{
  "connectionId": "string"
}
```

**&#x20; Step 3.   &#x20;**&#x46;rom within your PROCESIO Workspace, in the process listing tab, obtain the process ID for the process that you intend to run (hit the copy to clipboard button).

Additionally we will need the name of the Workspace under which the process was created. In the example below the name of the Workspace is: "Demo".

![](/images/docs/wO-xQyjyN0vxLHMqD6mqU_demo.png)

### Running a process from Postman using API keys

- Select Post Method mention the base URL and the endpoint
  - **Base URL**: https\://webApi.procesio.app
  - **Endpoint**: /api/Projects/e6bb2d64-2f96-4e43-a84b-b25a08c0764d/run (includes the Process ID)
- In Postman from Headers
  - **include key** & **key name**
  - **value** & **key value**
  - **workspace** & **workspace name** (example above "Demo")

![](/images/docs/Tx5QwCtrJ8199TWCAEhXs_postman.png)

- In Postman from Body, raw
  - include: **\&#123;  "connectionId": "string"&#125;**
  - Select **JSON**

![](/images/docs/eBrEDB-JcCzSNuinh0oal_postman2.png)

5\) From Postman hit the **Send&#x20;**&#x62;utton in order to trigger the process for which you configured the request.
