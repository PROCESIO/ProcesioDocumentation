### **Moving multiple actions together inside the canvas**

1. Hold down the **CTRL** key and **click** every action that you want to move.
2. Release the **CTRL** key and just **drag&#x20;**&#x74;he group like you would drag a single action.

![](/images/docs/KGF8L7F6qnjq_toxJrHgj_multi-select.gif)

### **Spawning an action on canvas without dragging it**

- **Double-click** the action you want to spawn and a new one will show up in the middle of the canvas.

![](/images/docs/wcbUSSaLOh0UOm_b5uzB7_spawner.gif)

### **Create and assign a variable in one step**

![](/images/docs/p4AVGBlbBxjbqeai-53yj_auto-assign.gif)

### **Full-screen code editor**

1. **Click** inside the code editor.
2. Use the **ESC&#x20;**&#x6B;ey to toggle the full-screen on and off.

![](/images/docs/wKycHFNX2KANp2zhJVsNl_fullscreener.gif)

### **Editing process properties faster**

- You can **double-click** on the process name inside the canvas to edit the process properties.

![](/images/docs/i1eyhIRWVVjPwBo3yPlxt_fast-edit.gif)

### **Grouping processes in the Process Designer**

- Use <span style={{color:'#2166ae'}}>Group</span><span style={{color:'#fcb900'}}>/</span><span style={{color:'#3b9f0f'}}>Name</span> pattern when naming your processes to group them.

![](/images/docs/OaqfOT2yiE_JOtmTQRdnt_group-process.gif)
