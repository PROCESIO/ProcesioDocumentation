*Every attribute listed in this guide is an attribute in the element's data model within the Procesio Forms feature. All attributes can be accessed and modified programmatically at runtime — including onLoad, onChange, onClick, and any other event available in Procesio Forms. This enables dynamic, data-driven form behavior without requiring custom code outside the platform's event system.*
