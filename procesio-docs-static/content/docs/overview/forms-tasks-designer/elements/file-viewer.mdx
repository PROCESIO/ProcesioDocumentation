The File Viewer element provides an inline or modal document preview experience directly within a form. Rather than forcing users to download a file to inspect it, the File Viewer renders the document in place — supporting file types like PDF — with customizable messaging for loading, empty, and error states.

The src attribute accepts a URL, base64 string, or file reference, making it easy to display files uploaded earlier in the same form, documents retrieved from a process, or static reference materials. The viewer can render inline (embedded directly on the form surface) or in a modal popup — giving you control over how much space the preview consumes.

## Key Use Cases

•        Displaying a contract or policy PDF for the user to review before signing or approving

•        Showing a document generated by a Procesio process (e.g., a quote or invoice) for review

•        Previewing a file immediately after it is uploaded in a File Upload element

•        Embedding reference materials (SOPs, specifications) alongside the form fields that reference them

•        Using modal mode to keep the form layout compact while still offering a full-screen view

## Runtime API — Attributes in Procesio Forms

*All attributes below are accessible and modifiable at runtime via the element's data model in Procesio Forms.*



| **API Name**           | **Description**                                     | **Type**     | **Default**          | **Notes**            |
| ---------------------- | --------------------------------------------------- | ------------ | -------------------- | -------------------- |
| **label**              | Display label for the file viewer                   | string       | —                    | —                    |
| **tooltip**            | Help text shown alongside the label                 | string       | —                    | —                    |
| **visible**            | Controls element visibility                         | boolean      | true                 | —                    |
| **src**                | File source — URL, base64 string, or file reference | string\|file | —                    | Updatable at runtime |
| **loadingviewmessage** | Message shown while the file is loading             | string       | loading              | —                    |
| **openviewmessage**    | Message shown as a prompt to open the viewer        | string       | click to open        | —                    |
| **emptyviewmessage**   | Message shown when no file source is set            | string       | field source not set | —                    |
| **failedviewmessage**  | Error message shown if loading fails                | string       | file is invalid...   | —                    |
| **allowedfiletypes**   | Permitted file types for display                    | string\[]    | pdf                  | —                    |
| **viewertype**         | How the viewer is presented                         | string       | —                    | inline, modal        |
| **name**               | Unique element name                                 | string       | —                    | —                    |
| **id**                 | Unique element ID                                   | string       | —                    | —                    |
