# Overview

This event allows you to manipulate and transform data between forms. You can define how input values from one form are stored, adjusted, or passed into another form or into internal variables. This ensures data consistency, reduces manual input, and allows dynamic control over workflows.

# Troubleshooting

If you encounter issues with your form events:

1. Check the event logs for error messages and details.
2. Verify the data mapping configurations.
