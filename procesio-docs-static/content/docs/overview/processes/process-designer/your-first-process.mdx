We're going to build a very basic process that will answer the following questio&#x6E;**&#x20;"Is it the weekend yet ?"**

Hit **Create a process** and let's start building !

![](/images/docs/yzaPTjvRRr3o8K7K8tFUN_dash-clean.png)

As a first step let's rename our process.

![](/images/docs/MAXPQozjrAp1nVln2xHAA_rename-examplee.gif)

<Callout type="info">
By providing a good name for our process we ensure anyone can understand what it does.&#x20;
</Callout>

### **Thought process**

- *What do we need to do exactly?*

We need to find out if the current date is **Saturday** or **Sunday**.

- *Sounds good, how should we do it?*

We should think about this process as a two-part job, first we **get the date** and then we **check if it's on a weekend** and interpret the result.

- *OK, but we don't want to input the date manually, right?*

Yes, we'll get the current date **automatically** so that we won't have to input it manually on every run.

- *And how can we do that?*

Let's check the **Platform Actions** section first to see if we can find an action that can get the current date.

- *There seems to be a&#x20;****DateTime****&#x20;folder so let's look over there for now*

![](/images/docs/xKp6QWsUkmSItzRnkqYPG_finding.gif)

**Today** seems like a good fit, let's check the [documentation page](https://docs.procesio.com/how-to/today) to learn how to use it.

![](/images/docs/JbF-CIwV760sJsjxwn1jQ_docs.gif)

- *Now we know how the&#x20;****Today****&#x20;action works, how do we proceed ?*

We'll declare an **Output** variable called **CurrentDate** of type DateTime which will hold the current date provided by the **Today** action.&#x20;

![](/images/docs/kP0RPQvyeKDTFPa9OaWVH_variabled.gif)

After this we'll connect the action to **Start** and **Stop** and make sure we've got a valid flow by clicking the **Validate** button and making sure no more errors show up.

![](/images/docs/M1LDz1SYS07dH-B4WBgNj_validatered.gif)

When the process becomes valid, we'll hit **Run** to see if the **CurrentDate** variable has the right value.

![](/images/docs/Vq5S1ZgiUxAYJ42fpJCPV_runtiming.gif)

- *Now we got the date... How do we figure out if it's on a weekend ?*

The thought process should be similar. We check the **DateTime** folder again to see if anything fits our needs.

- ***Is Weekend****&#x20;seems promising so let's check its&#x20;*[documentation page](https://docs.procesio.com/how-to/is-weekend)*.*

![](/images/docs/HtTTU2MrfHxYWE2p7Mxzv_wekeendd.gif)

Now that we found the perfect action, lets use it in our process to implement the weekend check. We'll declare another **Output** variable called **IsWeekend** of type Booleanwhich will hold the logic value of `Is CurrentDate on a weekend ?`

![](/images/docs/ocrRZpPvHxZ3kYOnhvbsf_variablesss2.gif)

After this, we chain the **IsWeekend&#x20;**&#x61;ction to the **Today&#x20;**&#x61;ction and make sure our flow is valid.

![](/images/docs/duzIwQ43-0s-yveBSB9z9_partial.gif)

Now we're done, let's see how **your first process** runs now 😁

![](/images/docs/MOeZ3lembHhWubwxZbBHp_partials.gif)

- *Somehow I don't really like the&#x20;****true****&#x20;and&#x20;****false****&#x20;stuff, is there a way to make is show messages?*

It's a bit more complex but sure, let's make it show **'Yes'** instead of **true** and **'No'** instead of **false**.

We'll declare a third **Output** variable called **Message** of type String to store the message.

![](/images/docs/pcVSl_-faUpAc78roIGMq_third.gif)

To convert the value of **IsWeekend** to a message we will use the following actions:

1. [Decisional](https://docs.procesio.com/how-to/decisional) ➜ to choose which message to map based on the value of **IsWeekend**
2. [Map Process Data](https://docs.procesio.com/how-to/map-process-data) ➜ to map the right message to the **Message&#x20;**&#x76;ariable
3. [Join](https://docs.procesio.com/how-to/join) ➜ to merg&#x65;**&#x20;Decisional&#x20;**&#x62;ranches&#x20;

Let's build the flow now and make sure it's valid.

![](/images/docs/zQKrojshz01CC7jjYFFsy_finalsssddd.gif)

And there you have it, lets now run it to see how it works.

![](/images/docs/jKeCbuDpJJV_X08Pp5WLj_finalrun.gif)
