PROCESIO allows you to work with documents by creating your own templates according to your requirements. The **Document Designer** gives you the ability to choose how the data is presented in your flows and with the use of variables you can create dynamic documents.

&#x20; 1\.    Navigate to the **Document Designer** section and click **Create new document**.

![](/images/docs/GJ5g9wsW5GBBeXXf8dMXc_image.png)

&#x20; 2\.     Create the variables you wish to use by clicking **+ Create new variable**.

&#x20; 3\.    Add the text and variable to your document. The variable can be added by clicking on it in the **Template variables** section.

![](/images/docs/5ChUb8mNOcgxTdvuqq2zA_image.png)

&#x20; 4\.    Click **Save Document**.

![](/images/docs/BJnxLFO7Gln0Zo3eMDusX_image.png)
