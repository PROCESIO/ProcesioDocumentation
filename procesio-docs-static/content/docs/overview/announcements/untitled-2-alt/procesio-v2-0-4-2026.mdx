# Release Notes — Platform Updates

This release brings new data visualization capabilities, performance improvements, and several user experience updates across the PROCESIO platform.

## Form Charts

We introduced **Form Charts**, a new feature that allows you to visualize form data directly inside PROCESIO.

With Form Charts, you can turn submitted form data into charts, making it easier to analyze responses, identify trends, and monitor collected information.

Read more in the documentation:
https\://docs.procesio.com/charts



![](/images/docs/sPgdKRvsM3qS9x_pbs2y0_image.png)

## Analytics Configuration Update

For **Business Workspaces**, Analytics is now **disabled by default** for newly created Subworkspaces.

Analytics can still be enabled from the **Subworkspace Configuration** settings whenever needed. This update gives workspace administrators more control over how analytics are configured across their workspace structure.

## Frontend Improvements

We made several frontend improvements to enhance usability and flow visibility.

### Current Action Highlight

The currently running action is now highlighted more clearly during execution, making it easier to understand where a process is in real time.



![](/images/docs/95_r9bJbpyD41vjBNHp5u_image.png)

### Schedule Start Date Fix

We fixed the schedule date behavior so users can now set the desired start date when configuring schedules.

This improves flexibility when creating scheduled executions and ensures schedules start when expected.

## Search Performance Improvements

Search has been improved on both the frontend and backend.

Search results should now load faster, making it easier to find resources, flows, and configurations across the platform.

## Login and Sign-Up Redesign

The **Login** and **Sign-Up** pages have been redesigned with a cleaner and more modern interface.

This update improves the onboarding experience and provides a more consistent visual experience across the platform.



![](/images/docs/uIqPnC4Ubypt7UV9sYrUf_image.png)

## Summary

This release includes:

- New **Form Charts** documentation and functionality
- Analytics disabled by default for Business Workspace Subworkspaces
- Improved current action highlighting
- Fixed schedule start date configuration
- Faster search experience
- Redesigned Login and Sign-Up pages

We will continue improving the PROCESIO experience based on your feedback.
