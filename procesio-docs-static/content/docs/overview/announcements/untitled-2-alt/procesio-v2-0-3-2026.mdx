# 🚀 Smarter Forms, Better DateTime Control & Enhanced Platform Experience

We’ve just rolled out a powerful new update packed with usability improvements, smarter controls, and enhanced platform capabilities. Here’s what’s new:

## ✨ Headless [Chat](../spaces/SPACES/Overview/Chat.md) for Forms

Forms just got more powerful.

You can now use **Chat Control in headless mode**, giving you full flexibility over how conversational experiences are embedded and displayed.
Build cleaner interfaces, control the UX, and create seamless chat-driven workflows — your way.

## 🕒 [Date-time Input](&lt;../spaces/SPACES/Overview/Date-time Input.md>) Control – Smarter & More Flexible

Managing dates and time is now easier than ever:

- **Inline View** for a smoother, more intuitive selection experience
- **Available & Disabled Time Intervals** to precisely control scheduling logic

More control. Fewer errors. Better user experience.

## 🔐 Improved Authentication with Cookies

We’ve enhanced authentication by introducing **cookie-based session handling**, resulting in:

- More stable sessions
- Improved security
- A smoother overall experience

Less friction. More reliability.

## 🔎 Extended “Test Connection” Capabilities

You can now test **more actions directly from Test Connection**, making setup and troubleshooting faster and more accurate.

Configure with confidence.

## 🛠 Enhanced “String to DateTime” Action

We’ve improved parsing accuracy and format handling, making date conversions more reliable across different input patterns.

Cleaner data. Stronger workflows.

## 📊 Filter History Instances by Date

Need to investigate a specific timeframe?
You can now **filter History Instances by date**, making audits and troubleshooting significantly easier.

Find what matters — faster.

This release focuses on giving you more control, better precision, and a smoother experience across the platform.

Let us know what you think!
