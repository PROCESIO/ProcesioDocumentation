## Overview

The `Split` action divides a string based on the occurences of a given separator.

![](/images/docs/rGeFfZD4M4VpUuy9hPZvR_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to split.
- **\[ I ] Input separator:** <span style={{color:'#1c417e'}}>String</span> to split based on.
- **\[ O ] Result**: **List\&lt;**<span style={{color:'#1c417e'}}>String</span>**>** containing the substrings resulted from dividing the input string based on the separator.
