## Overview

The `Split` action divides a string based on the occurences of a given separator.

![](/images/docs/rGeFfZD4M4VpUuy9hPZvR_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** String to split.
- **\[ I ] Input separator:** String to split based on.
- **\[ O ] Result**: **List\&lt;**String**>** containing the substrings resulted from dividing the input string based on the separator.
