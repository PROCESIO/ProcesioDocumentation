## Overview

The `Trim End` action removes whitespaces from the end of a string.

![](/images/docs/A1DF2tu8ZlVEGGQg1T9AJ_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to be trimmed.
- **\[ O ] Result**: <span style={{color:'#1c417e'}}>String</span> representing the input after trimming whitespaces from the end.
