## Overview

The `Starts With` action checks if a string starts with another string.

![](/images/docs/ZQhlccnS1WGT8Go1AcOPo_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to search in.
- **\[ I ] Input string - start:** <span style={{color:'#1c417e'}}>String</span> to start with.
- **\[ O ] Result**: <span style={{color:'#7239e3'}}>Boolean</span> representing the logic value of `Does the Input string start with the Input string - start value ?`
