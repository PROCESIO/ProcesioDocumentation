## Overview

The `Trim Start` action removes whitespaces from the beginning of a string.

![](/images/docs/qW4J3AUaDblCuBkKN3qKK_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to be trimmed.
- **\[ O ] Result**: <span style={{color:'#1c417e'}}>String</span> representing the input after trimming whitespaces from the start.
