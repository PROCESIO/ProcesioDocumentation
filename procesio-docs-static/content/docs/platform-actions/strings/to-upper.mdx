## Overview

The `To Upper` action converts a string to uppercase.

![](/images/docs/RQdU_uSbvEkXjKZSe20fS_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to be converted.
- **\[ O ] Result**: <span style={{color:'#1c417e'}}>String</span> representing the input after conversion to uppercase.
