## Overview

The `To Upper` action converts a string to uppercase.

![](/images/docs/RQdU_uSbvEkXjKZSe20fS_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** String to be converted.
- **\[ O ] Result**: String representing the input after conversion to uppercase.
