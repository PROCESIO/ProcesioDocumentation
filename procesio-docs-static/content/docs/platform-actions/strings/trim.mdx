## Overview

The `Trim` action removes whitespaces from both ends of a string.

![](/images/docs/wtpvIflqUDOEjvOrByALF_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to be trimmed.
- **\[ O ] Result**: <span style={{color:'#1c417e'}}>String</span> representing the input after trimming whitespaces from both ends.
