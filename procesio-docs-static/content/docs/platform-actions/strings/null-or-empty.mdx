## Overview

The `Null Or Empty` action checks if a string is null or empty.

![](/images/docs/Sb11D_l6AcnPg-GjOb1Fu_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** String to check.
- **\[ O ] Result**: Boolean representing the logic value of `Is the input string null or empty ?`
