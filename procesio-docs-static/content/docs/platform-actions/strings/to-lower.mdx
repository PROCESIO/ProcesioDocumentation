## Overview

The `To Lower` action converts a string to lowercase.

![](/images/docs/h1KAxCyqQaof0i4SfHGhw_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to be converted.
- **\[ O ] Result**: <span style={{color:'#1c417e'}}>String</span> representing the input after conversion to lowercase.
