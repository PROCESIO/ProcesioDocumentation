## Overview

The `To Lower` action converts a string to lowercase.

![](/images/docs/h1KAxCyqQaof0i4SfHGhw_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** String to be converted.
- **\[ O ] Result**: String representing the input after conversion to lowercase.
