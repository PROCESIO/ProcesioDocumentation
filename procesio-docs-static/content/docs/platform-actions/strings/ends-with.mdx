## Overview

The `Ends With` action checks if a string ends with another string.

![](/images/docs/L0T_9gMFwUXd1zT_yG7JN_image.png)

## Inputs & Outputs

- **\[ I ] Input string:** <span style={{color:'#1c417e'}}>String</span> to search in.
- **\[ I ] Input string - end:** <span style={{color:'#1c417e'}}>String</span> to end with.
- **\[ O ] Result**: <span style={{color:'#7239e3'}}>Boolean</span> representing the logic value of `Does the Input string end with the Input string - end value ?`
