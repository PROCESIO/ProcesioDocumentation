The **Numerical&#x20;**&#x66;older contains actions that can be used while working with numbers:

1\. **Add operation** ➜ Addition operation for two objects.

2\. **Subtract operation** ➜ Subtraction operation for two objects.

3\. **Modulo operation** ➜ Modulo operation for two integers.

4\. **Divide operation** ➜ Division operation for two objects.

5\. **Round Number&#x20;**➜ Round a number to the specified number of decimal places.

6\. **Round Up Number&#x20;**➜ Round up a number to the specified number of decimal places.

7\. **Round Down Number** ➜ Round down a number to the specified number of decimal places.
