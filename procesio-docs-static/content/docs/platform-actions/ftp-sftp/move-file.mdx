The action **Move file**, enables you to move a file from one folder to another, or from one FTP server to another.

### **FTP connection****&#xFEFF;****&#xFEFF;****&#xFEFF;**

For our example we used [Rebex](https://www.rebex.net/support/tutorial/ftp/).

&#x20; **Step 1**:    Go to the **Credentials Manager** section and click **Add New**.

&#x20; **Step 2**:    Give your credential a **Name\*** and **Description&#x20;**(optional) and click **Next step**.

![](/images/docs/PQ2tRh609R0DcOE95q7LU_image.png)

&#x20;    **Step 3**:    Configure the FTP / sFTP connection as needed:

- **Port number\*** - the port number used by the server.
- **Protocol type** - the protocol used depending on the encryption.
- **Directory path** - the entry point of the server.
- **Authentication method** - here you have the option to use a basic username and password authentication or a user key authentication.

![](/images/docs/Lprp00FxIIb1N8ILJbZcJ_image.png)

&#x20;   **Step 4**:    Click **Test connection** and if successful click **Save**.

### **Create the process****&#xFEFF;****&#xFEFF;****&#xFEFF;**

&#x20; **Step 1**:    From the **Process Designer** section select **Create new**.

&#x20; **Step 2**:    Give you process a name.

&#x20; **Step 3**:    Drag the action **Move File** on the canvas and link the action inside you process.

![](/images/docs/6Hu1PyH5vYwwY0XQn8jNY_image.png)

&#x20; **Step 4**:    Create the <span style={{color:'#2166ae'}}>&lt;%result%&gt;</span> Boolean variable of type output.

![](/images/docs/mi1Maf84CLcn2qopzAxc8_image.png)

&#x20;   **Step 5**:    Click the **Move File** action to access its configurations.

- **Node name\*** - gives you the option to change the actions name.
- **From FTP server\*** - here we will select the connection created in the previous section.
- **From Folder** - folder path to the file.
- **File name** - the name of the file that will be moved.
- **To FTP server\*** - here we will select the connection to the server we wish to move the file to.
- **To Folder** - new folder path to the file.
- **Result\*** - here we will add the variable created at **Step 4**.

![](/images/docs/Er0TsFnRgt4c8H0MxwsHt_image.png)

&#x20;   **Step 6**:    **Save**, **Validate&#x20;**&#x61;nd **Run&#x20;**&#x74;he process.

&#x20;   **Step 7**:    Click **Check Instance**.
