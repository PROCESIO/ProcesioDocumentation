Procesio can interpret HTML code an generate PDF files the said code.

**How to**[﻿](https://docs.procesio.com/duplicate-identification#lm-how-to)**&#xFEFF;**[﻿](https://docs.procesio.com/list-filter-simple#oy-how-to)**&#xFEFF;**[﻿](https://docs.procesio.com/list-filter#s9-how-to)**&#xFEFF;**

&#x20;   Step 1.    [Create a new process](https://docs.procesio.com/creating-a-basic-process) and give it a name.

![](/images/docs/wt7abaoEoYapwxxlsz8dZ_image.png)

&#x20;     Step 2.    Drag the **HTML to PDF** action to the canvas.

&#x20;     Step 3.    Create the &lt;%OutputFile%&gt; output variable of type File.

![](/images/docs/DX_RaYzF7NPp3pKeWgwDJ_image.png)

&#x20;     Step 4.    Click the **Configure&#x20;**&#x62;utton inside the action and use the following configuration.

- **Html string** - The string containing HTML input.
- **File name** - Name of the file to be downloaded.
- **File output** - Returns the generated PDF in a variable of type file. We will add here the variable we created at the previous file.

![](/images/docs/41JqaUanqdcO_QvjG8fG2_image.png)

&#x20;     Step 5.    **Save**, **Validate** and **Run** the process.

&#x20;     Step 6.    Click **Check instance**.

&#x20;     Step 5.    In the output section you have the option to download the file.

![](/images/docs/KSmibXS617qZ66E2j9-_o_image.png)

![](/images/docs/Gvl4U5-Pzs9XK2kO0lUxT_image.png)
