Procesio allows you to work with several types of files, one of the most popular files used within the business world is XLSX, a Microsoft Excel Open XML Format Spreadsheet file.

### **&#x20;Scenario**

The user wishes to use the process to export data in the form a excel file.

### How to

&#x20;   Step 1.    [Create a new process](/docs/overview/processes/process-designer/your-first-process) and give it a name.

![](/images/docs/KR8BuERsoaZ_acPcusgKZ_image.png)

&#x20;    Step 2.    [Add the needed variables](/docs/overview/processes/process-designer/working-with-process-variables).

For the process used in our example we will require the following variables:

- <span style={{color:'#2166ae'}}>&lt;%data%&gt;</span> - the input list\&lt;Object> variable that will store the JSon.
-
  <span style={{color:'#2166ae'}}>&lt;%export%&gt;</span> - the output variable of type File.

![](/images/docs/AmPJwAKSxL_Df_njqxYSi_image.png)

&#x20;    Step 3.   Click the **Export to XLSX** action and add the following configuration.

- Object to export\* - will hold the <span style={{color:'#2166ae'}}>&lt;%data%&gt;</span> variable.
-
  Column delimiter - will ask for the type of delimiter used in the newly created file.
-
  Export header - the checkbox gives us the option to also export the headers.
-
  File name\* - will be the name of the exported file, the extension will be added by Procesio.
-
  Output file\* - will contain the fil variable we created  in Step 2.

<Callout type="info">
The **File name\*** should be the same as the **Output file\*** name. Check the screenshot below.
</Callout>

![](/images/docs/LYIy-JCK4SheZFo7WC_xm_image.png)

&#x20;   Step 4.   **Save**, **Validate&#x20;**&#x61;nd **Run&#x20;**&#x74;he process.

&#x20;   Step 5.   You will be prompted to add the data for our input variable. The input will be of type JSon. Click **Run**.
