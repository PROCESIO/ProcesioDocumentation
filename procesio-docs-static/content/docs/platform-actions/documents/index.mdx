After creating a process you will have the ability to use the drag and drop functionality to add on your canvas actions from the Platform actions section that will allow you to work with files, this can be done in conjunction with the [Document Designer](/docs/overview/processes/document-designer/document-template-creation).

The **Documents** folder contains actions that allow you to create and work with files inside your processes. 

&#x20; 1\.    **Generate Document** - Generate a document using a Document Designer template.

&#x20; 2\.    **HTML to PDF &#x20;**- Converts HTML input to a PDF file.

&#x20; 3\.    **Export to CSV&#x20;**-**&#x20;&#x20;**&#x45;xports a list (of primitives/simple or of data model) to a .csv file, with export options.

&#x20; 4\.    **Export to XLSX&#x20;**- Exports a list (of primitives/simple or of data model) to a .xlsx file.

&#x20; 5\.    **Write to file -&#x20;**&#x57;rites any object to a File that the user can download in any format that he wants

&#x20; 6\.    **Write list to file -&#x20;**&#x57;rites any list\&lt;object> to a File which the user can download in any format that he wants

&#x20; 7\.     **Read file -&#x20;**&#x52;eads the content in a file (ex. you have a document which contains some text and you want to make a variable of type string with the document's content)

&#x20; 8\.    **File to Base64 -&#x20;**&#x43;onverts the content of a File variable to a Base64 string

&#x20; 9\.   **Archive** sub-folder - contains actions that allow you to compress and decompress data into different archive types.
