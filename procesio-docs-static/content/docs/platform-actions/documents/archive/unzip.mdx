Use the **Unzip** action to extract compressed files from a zip file.



### How to configure the Unzip action?

1\.   Create a process and give it a name.

2\.    Drag the **Unzip** action to the canvas and link it to the other actions.

![](/images/docs/wKlhTvU5Wk8mvxNeW-7kG_screenshot-2022-08-25-at-153414.png)

3\. Create the variables needed for the configuration of the action, and then add them to the configuration panel:

![](/images/docs/O5JJedf3YxFpHvlQtouxG_screenshot-2022-08-25-at-154100.png)

1. Fill in configuration details:

**Archive:** the variable for zip file you want to decompress;&#x20;

&#x20;                Mandatory: yes

**File list:&#x20;**&#x74;he variables for the files that will result fro decompressing the archive

&#x20;                Mandatory: yes

2\. Save, Validate and Run the process.

3\. The process will ask for an input file (the archive).

4\. Click **Run**

5\. Click **Check Instance&#x20;**&#x74;o view the results.

![](/images/docs/7LECZfRRBh7s0ELe0ePqQ_screenshot-2022-08-25-at-154218.png)

Click on the download icon next t&#x6F;**&#x20;Output&#x20;**&#x61;nd the files extracted from the zip archive will be downloaded.&#x20;
