The **Concatenate&#x20;**&#x61;ction allows you to create a list by merging the data of two other lists.

<Callout type="warn">
Concatenate was previously called Concatenate Lists.
</Callout>



### **Scenario**

As a PROCESIO user, I want to have an action that concatenates 2 lists (the lists can be simple or DM). The action outputs a new list containing values from both input lists.

### **How to**

The example requires a process with just the **Concatenate** action with the following configuration.

**&#x20;   Step 1.   &#x20;**[Create a new process](/docs/overview/processes/process-designer/your-first-process) and give it a name.

&#x20;**&#x20;  Step 2.   &#x20;**&#x44;rag the **Concatenate** action to the canvas and link it to the other actions.

![](/images/docs/LVqB-8dBkjUB72O9x9CYt_image.png)

**&#x20;  Step 3.   &#x20;**[Add the needed variables](/docs/overview/processes/process-designer/working-with-process-variables).

![](/images/docs/FOGZxbDHEb9gG5voOur9u_image.png)

**&#x20;  Step 4.   &#x20;**&#x43;lick the **Concatenate&#x20;**&#x61;ction to access its configurations and add the variables as follows.

![](/images/docs/swsZLMHJjC5FTYXqVwWjz_image.png)

**&#x20;  Step 5.    Save** and **Validate&#x20;**&#x74;he process.

**&#x20;  Step 6.    Run&#x20;**&#x74;he process.

&#x20;**&#x20;  Step 7.   &#x20;**&#x43;lick **Check instance** to see the results in the output section.
