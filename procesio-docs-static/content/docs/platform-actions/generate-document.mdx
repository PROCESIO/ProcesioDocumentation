After creating the [template](https://docs.procesio.com/document-template-creation) you have the ability to generate documents in your processes.

&#x20; 1\.    Navigate to the [Process Designer](https://docs.procesio.com/creating-a-basic-process) section and create your process.

&#x20; 2\.    Give it a name.

&#x20; 3\.    Drag the **Generate Document** action to the canvas.

![](/images/docs/nmeiefzUvY3AM3fElZaXb_image.png)

&#x20; 4\.    Create the [variables](https://docs.procesio.com/working-with-process-variables).

- **fileout&#x20;**➜ the output variable of type File. &#x20;
- **htmlstring** ➜ the output variable of type String.&#x20;
- **Name** ➜ the input variable of type String.

![](/images/docs/KrXNIFDicVNgys03wwWoE_image.png)



&#x20; 5\.    Click the **Generate Document** action to access its configurations.

- **Select document template\*&#x20;**- select the template we created [here](https://docs.procesio.com/document-template-creation).&#x20;
- **Save document as&#x20;**- the document will be saved s a PDF.
- **File name&#x20;**- choose the name of the file we will create.
- **Document output -&#x20;**&#x61;dd the file variable created in the previous step.
- **HTML string -&#x20;**&#x61;dd the string **htmlstring** variable created in the previous step.

![](/images/docs/O_l-HLzsl5ErfTCMdsvdA_image.png)

&#x20; 6\.    Click **Map Document Data&#x20;**&#x61;nd map the input variable **Name** created in the process to the **Name** variable used in the template (they do not need to have the same name).

![](/images/docs/x-pyBGITBEVflCYZVXBPT_image.png)

&#x20; 7\.    **Save**, **Validate** and **Run** the process.

&#x20; 8\.    The process will ask for the input variables value. &#x20;

![](/images/docs/N_6SmQ_YMbwJXzmyyT5Zr_image.png)

&#x20; 9\.    Click **Run** again and than click **Check instance**.

&#x20; 10\.    If you wish to download the the file or the html output you just need to click the download button.&#x20;

![](/images/docs/LbYMjo5gUlU6owQGiucm2_image.png)
