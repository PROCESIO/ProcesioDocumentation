The **Converter** folder contains utility actions:

&#x20; 1\.    **FromBase64&#x20;**➜ Decodes a base64 string.

&#x20; 2\.    **ToBase64&#x20;**➜ Encodes a string to base64.

&#x20; 3\.    **ToString&#x20;**➜ Convert any object to a string.

&#x20; 4\.    **Json To Xml&#x20;**➜ Convert JSON to XML.

&#x20; 4\.    **Xml To Json&#x20;**➜ Convert XML to JSON.

&#x20; 4\.    **Encrypt to AES256&#x20;**➜ Encrypt a string using AES256.

&#x20; 4\.    **ecrypt from AES256&#x20;**➜ Decrypt a string encrypted using AES256.
