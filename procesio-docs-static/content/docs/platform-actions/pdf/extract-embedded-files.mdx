The **Extract Embedded Files** action extracts embedded files from a .pdf.

![](/images/docs/boPU9zfNHyfieXhYqS5Dl_image.png)

### How to configure the Extract Embedded Files action?

&#x20;    1\. Create a process and give it a name.

&#x20;    2\. Drag the **Extract Embedded Files** action to the canvas and link it to the other actions.

&#x20;    3\. Create the variables needed for the configuration of the action, and then add them to the configuration panel:&#x20;

### Inputs & Outputs

- **\[ I ]** **Pdf file**: Filerepresenting the input pdf file that contains embedded files.
- **\[ O ] Embedded files**: **List\&lt;**File**>** representing the list of the embedded files extracted from the input pdf.

![](/images/docs/BlICZK7qdn1KPePSCYY9f_image.png)

&#x20;    5\. You will be prompted to add the file for the input variable. **Drag** or **upload&#x20;**&#x74;he file and click **Run**.
