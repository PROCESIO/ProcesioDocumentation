### Overview[﻿](https://docs.procesio.com/write-to-file#yJA0T)﻿

Use this action to merge two .pdf files into one.&#x20;

<Callout type="info">
The order of the pages will be the same as the order of the input files.
</Callout>

### How to configure the Merge files action?[﻿](https://kb.procesio.com/regex-replace#qd-how-to)﻿[﻿](https://docs.procesio.com/merge-lists#q-how-to)﻿[﻿](https://docs.procesio.com/write-to-file#-s0Vi)﻿

Step 1. Create a process and give it a name.

Step 2. Drag the **Merge files** action to the canvas and link it to the other actions.

![](/images/docs/aj0Bk8W5b3etopbNC6nCz_image.png)

Step 3. Create the variables needed for the configuration of the action, and then add them to the configuration panel:

![](/images/docs/5TbeWiKIMa09vZndR8vr8_screenshot-2022-09-21-at-153220.png)

**First PDF**: you need to input a File which contains a pdf&#x20;

**Second PDF**: you need to input a File which contains a pdf&#x20;

**Merge Result**: need to add a File variable which is the output with the merged .pdf with first and second pdf files.&#x20;

<Callout type="info">
When downloaded, the Output file name will be: FirstFileName\_SecondFileName.pdf, where:

FirstFileName - is the name of the first PDF file&#x20;

SecondFileName - is the name of the second PDF file
</Callout>

Step 4. **Save**, **Validate&#x20;**&#x61;nd **Run&#x20;**&#x74;he process.

Step 5. You will be prompted to add the file for our input variable. Click **Run**.

Step 6. To verify the process click **Check instance**.

Step 7. In the **Output&#x20;**&#x73;ection, you can see the output variable containing the 2 PDF files merged into one.
