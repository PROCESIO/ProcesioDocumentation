## Overview

The `Acroform Insert Image` action inserts an image into a PDF file to a location where a marker of type Acroform Field Name is set. The image will be inserted in the boundaries of the Acroform Field.

![](/images/docs/A5hAfxm3G6ITSxj0hYsrL_image.png)

### Parameters

### Inputs

- **PDF in** (`file`). The input PDF. It must contain an Acroform Field.
- **Marker name** (`string`). The name of the Acroform Field.
- **Image** (`file`). The image to be added where the Acroform Field is defined.

### Outputs

- **PDF in** (`file`): The output.

## Important Notes

- Chaining multiple actions is possible only by starting with the highest number marker before others.
  - Example: `supplier.signature.3``  →  ``supplier.signature.2``  →  ``supplier.signature.1`
