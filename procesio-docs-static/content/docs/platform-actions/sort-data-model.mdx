The **Sort data model&#x20;**&#x61;ction allows you to create a list by merging the data of two other lists.

### **Scenario**

As a PROCESIO user who created a data model to use in my project I need a way to sort the data by multiple criteria.

### **How to**

The example requires a process with just the **Sort data model** action with the following configuration.

**&#x20;   Step 1.    Create the&#x20;**[data model](/docs/overview/processes/data-model-designer/working-with-data-models) CompanyDetails who uses the data model Compan&#x79;**.**

*(Screenshot unavailable — image was removed from the original source.)*

*(Screenshot unavailable — image was removed from the original source.)*

**&#x20;   Step 2.   &#x20;**[Create a new process](/docs/overview/processes/process-designer/your-first-process) and give it a name.

&#x20;**&#x20;  Step 3.   &#x20;**&#x44;rag the **Sort data model** action to the canvas and link it to the other actions.

*(Screenshot unavailable — image was removed from the original source.)*

&#x20;**&#x20;  Step 4.   &#x20;**&#x43;reate the [variable](/docs/overview/processes/process-designer/working-with-process-variables) we will use to hold the input data, for simplicity we will provide a default value to it.

*(Screenshot unavailable — image was removed from the original source.)*

```json
{
  "id": "asfw2mki4",
  "Company": [
    {
      "city": "Berlin",
      "Company": "Rutherford Group"
    },
    {
      "city": "London",
      "Company": "Nevett"
    }
  ]
}
```

**&#x20;  Step 5.   &#x20;**&#x43;lick the **Sort data model** action to access its configurations and add the variables as follows. We will use a json configuration that will sort the city in ascending order and the company name in descending order.

*(Screenshot unavailable — image was removed from the original source.)*

```json
{
  "IdField":"id",
  "Sort":[
			{"Field":"Company.city","OrderType":"Asc"},
			{"Field":"Company.company","OrderType":"Desc"},
		]
}
```

**&#x20;     Step 6.    Save** and **Validate&#x20;**&#x74;he process.

**&#x20;     Step 7.    Run&#x20;**&#x74;he process.

&#x20;**&#x20;     Step 8.   &#x20;**&#x43;lick **Check instance** to see the results in the output section.
