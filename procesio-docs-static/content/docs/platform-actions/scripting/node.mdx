Th&#x65;**&#x20;Node** action is one of the actions at your disposal that allows you to execute **Javascript** code.

Module imports can be performed using CommonJS **require('module')** syntax only.

Use the **return&#x20;**&#x6B;eyword to set the output value. You can return either **Single Values&#x20;**&#x6F;r **List** **Values**, not both at same time.

**Timeout**: → <span style={{color:'#16c9c9'}}>Integer</span> representing execution time limit(when timeout limit is reached, the action will throw an error)
**Single result**: → Output that will be set for single value outputs.
**List result**: → Output that will be set for list value outputs.
**Error**: → <span style={{color:'#9c70ce'}}>String</span> representing errors resulted from the execution of the script (the output is set only when an error occurs, otherwise it’s **null**)

<Callout type="warn">
Something to consider before trying to use the **Node&#x20;**&#x61;ction to automate your use cases is that it does not support working with files.
</Callout>

### How to configure the Node action?

1. Create a process and name it.
2. Drag the Node action to the canvas and link it to the other actions.

![](/images/docs/rawNBVs7ny9GmOy4DkD-A_image.png)

&#x20;    3\. Create the variables needed for the configuration of the action, and then add them to the configuration panel.

&#x20;    4\. Add your **Javascript Code** to the editor.

&#x20;    5\. Configure the output variables which will store the results.

![](/images/docs/i0i01-yIBANxb1JgX4-0O_image.png)

&#x20;    6\. Save, Validate and run.

&#x20;    7\. Click **Checked Instance** to view the results. You will see that the outcome of the interpreted code is saved in the **result**  variable. The script ran without errors so the value of the **errorMessage&#x20;**&#x76;ariable is null.

![](/images/docs/kxhn4zX_HO1w2K_XNhwjv_image.png)

### Usage Examples:

**&#x20;    1. Timeout:**

> return (async () => \&#123;
>     await new Promise(resolve => setTimeout(resolve, 10000)) //This code waits for 10 seconds.
> &#125;)()

Obs: If you set the **Timeout&#x20;**&#x74;o 5 seconds, the action will throw an error at runtime.

**&#x20;    2. Scripts with Single outputs.**

<span style={{color:'#16c9c9'}}>Integer</span>

> var a = 1
> var b = 2
> return a + b

<span style={{color:'#935b0e'}}>Float</span>

> var a = 1.0
> var b = 2.5
> return a + b

<span style={{color:'#9c70ce'}}>String</span>

> var a = "aa"
> var b = "bb"
> return a + b

<span style={{color:'#ff6900'}}>Boolean</span>

> var a = 1
> var b = 2
> return a == b

<span style={{color:'#30c922'}}>Json / DataModel</span>

> var x = \&#123;
> &#x20; "user": \&#123;
> &#x20;   "id": 1,
> &#x20;   "name": "Someone",
> &#x20;   "height": 170.5
> &#x20; &#125;
> &#125;
>
> return x

<span style={{color:'#10a2f9'}}>DateTime</span>

> let d = new Date()
> return d

Obs: These should only set the Single output. Should work for all types.


**&#x20;    3. Scripts with List outputs.**

> let d = new Date();
> let da = new Date();
>
> return \[d, da]

Obs: These should only set the List output. Should work for all types, just like single values.
&#xA;**&#x20;    4. Basic module usage**

If the module does not support CommonJS **require**, we can’t use it

> var \_ = require('underscore');
>
> return \_.map(\[1, 2, 3], function(num)\&#123; return num \* 3; &#125;)

> const R = require('ramda');
>
> const greet = R.replace('\&#123;name&#125;', R.\_\_, 'Hello, \&#123;name&#125;!');
>
> return greet('Alice');

> let cheerio = require('cheerio')
> let $ = cheerio.load('\<h2 class="title">Hello world\</h2>')
>
> $('h2.title').text('Hello there!')
> $('h2').addClass('welcome')
>
> return $.html()

&#xA;**&#x20;    5. Syntax errors&#x20;**

> var x = 1
> retur x

Ob&#x73;**:** Syntax errors will only set the **Error&#x20;**&#x66;ield.
&#xA;**&#x20;     6. Throw behavior**

> var x = 1
>
> throw "Not on my watch"
>
> return x

Obs: Exception messages thrown using **throw “message“&#x20;**&#x73;yntax will show up on the **Error&#x20;**&#x66;ield. Only **Error&#x20;**&#x77;ill be set after execution.
&#xA;**&#x20;    7. Async / Await**

> const axios = require('axios')
>
> const findInDictionary = async (word) => \&#123;
> &#x20;   const response = await axios.get(\`https\://api.dictionaryapi.dev/api/v2/entries/en/\` + word)
> &#x20;   return response.data
> &#125;;
>
> return findInDictionary('dog')
