### Overview[﻿](https://docs.procesio.com/write-to-file#yJA0T)﻿

Use this action to return the length of a string.

### How to configure the Length action?[﻿](https://kb.procesio.com/regex-replace#qd-how-to)﻿[﻿](https://docs.procesio.com/merge-lists#q-how-to)﻿[﻿](https://docs.procesio.com/write-to-file#-s0Vi)﻿

Step 1. Create a process and give it a name.

Step 2. Drag the **Length** action to the canvas and link it to the other actions.

![](/images/docs/AUWj5KxdoLBnWtwMb77vv_image.png)

Step 3. Create the variables needed for the configuration of the action, and then add them to the configuration panel:

![](/images/docs/jJmV_ezjj2QsPcbBTxIUQ_screenshot-2022-09-21-at-165746.png)

**Input string**: you need to input the string variable

**Result**: This is the output of type Integer, representing the length of the input string

Step 4. **Save**, **Validate&#x20;**&#x61;nd **Run&#x20;**&#x74;he process.

Step 5. To verify the process click **Check instance**.

Step 6. In the **Output&#x20;**&#x73;ection, you can see the result:

![](/images/docs/ViaWQ2cQ4ac20oauOTbsv_screenshot-2022-09-21-at-165841.png)
