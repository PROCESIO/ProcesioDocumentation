**List ToString&#x20;**&#x77;ill allow you to create a string from a list that can contain different data types and can be created with the aid of a [data model](https://docs.procesio.com/working-with-data-models).

**How to**

The example requires a process with just the **List ToString** action with the following configuration.

**&#x20;  Step 1.   &#x20;**[Create a new process](https://docs.procesio.com/creating-a-basic-process) and give it a name.

**&#x20;  Step 2.   &#x20;**&#x44;rag the **List ToString&#x20;**&#x61;ction action to the canvas and link it to the other actions.&#x20;

![](/images/docs/0I21XAnGwQOtzQprJcZLO_image.png)

**&#x20;  Step 3.   &#x20;**&#x43;reate the [data model](https://docs.procesio.com/working-with-data-models).

![](/images/docs/vYoz01edO3YtvI_Qgsd-A_image.png)

<Callout type="info">
We can use different data types for our attributes, depending on your requirements.
</Callout>

**&#x20;  Step 4.   &#x20;**[Add the needed variables](https://docs.procesio.com/working-with-process-variables).

![](/images/docs/EFPJPJmTpf4BvfjvmiEJL_image.png)

- ****&lt;%inputList%&gt;**&#x20;-&#x20;**&#x74;he input variable of type list, created with the data model **book&#x20;**&#x63;reated at the previous step.&#x20;
- ****&lt;%Result%&gt;**&#x20;-&#x20;**&#x74;he output string variable that will hold the result of the **List Totring** action.

<Callout type="info">
The input list variable can be created without the aid of a data model, but it will only use one data type inside the list.
</Callout>

**&#x20;  Step 5.  &#x20;**&#x20;Click the **List ToString&#x20;**&#x61;ction to access its configurations.

![](/images/docs/oBcUfYS7HpliVXo27HVMP_image.png)

**&#x20;  Step 5.    Save** and **Validate&#x20;**&#x61;n&#x64;**&#x20;Run&#x20;**&#x74;he process.

&#x20;**&#x20; Step 6.   &#x20;**&#x45;ach time you will run the process you are expected to add the list you wish to process, if you entered a default value you can edit it at this point and clic&#x6B;**&#x20;Run**.

![](/images/docs/4rHLhZVbFcM5SPBjGqBcA_image.png)

```json
[
    {
        "book": "Wings of Fire",
        "author": "A. P. J. Abdul Kalam and Arun Tiwari",
		"year": 2012,
        "email": "author1@gmail.com"
    },
	    {
        "book": "The Hidden Kingdom",
        "author": "Tui T. Sutherland",
		"year": 2013,
        "email": "author2@gmail.com"
    }
]
```

**&#x20;       Step 7.   &#x20;**&#x43;lick **Check instance** to see the result in the outputted variable.

![](/images/docs/8-sozELWTTn4V9wDQYiIl_image.png)

As you can see the single value variable of type String will contain the input data.
