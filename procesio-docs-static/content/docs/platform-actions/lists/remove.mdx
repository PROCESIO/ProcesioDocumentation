## Overview

The `Remove` action allows users to remove a specified object from a given list.



![](/images/docs/Q_I79s__U0iTmaG6VA5EY_image.png)

### Parameters

### Inputs:

- List from which the object is removed (`list<object>`): This parameter represents the list from which the specified object will be removed. After the removal, this variable will contain the updated list.
- Input object to be removed (`object`): This parameter specifies the object that needs to be removed from the list.

### Outputs:

- List from which the object is removed (`list<object>`): This output provides the updated list after the specified object has been removed.
