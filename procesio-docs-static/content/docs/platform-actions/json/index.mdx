After creating a process you will have the ability to use the drag and drop functionality to add on your canvas actions from the Platform actions section.

The Json folder contains actions that can be used while working with Json inputs:

1\. [Json Mapper](/docs/platform-actions/xpath/mapper) - JSON Mapper enables you to extract a value from a JSON or a JSON array.

2\. [Json Mapper To List](/docs/platform-actions/xpath/mapper) - JSON Mapper enables you to extract a list of values from a JSON or a JSON array.
