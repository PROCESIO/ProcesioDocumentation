The **Extract List** action is one of the actions at your disposal to work with Json data. Extract List enables you to extract a list of values from a JSON or a JSON array.

The action will be able to query/extract data from a JSON input using a [JSONPath expression](https://goessner.net/articles/JsonPath/) and save it to a variable.

<Callout type="warn">
Extract List was previously called Json Mapper to list.
</Callout>



### Overview

The action is used to filter the Json data that we will import in our process for future use.

Input Field 1: “Input JSON” - Input your JSON object / Array or a data model.

Input Field 2:  “Query” - The Input Jquery expression for obtaining the desired key value of your Json input.

Output Field: "Output" - Returns a list of values corresponding to the JQuery expression.


**How to**

The variables need to be adjusted according to your expected result for each process.

**Example 1**

&#x20; 1\.    [Create a new process](/docs/overview/processes/process-designer/your-first-process) and give it a name.

&#x20; 2\.    Drag th&#x65;**&#x20;Extract List** action to the canvas and link it to the other actions.

3\.    [Add the needed variables](/docs/overview/processes/process-designer/working-with-process-variables).

![](/images/docs/goZLbUDlHQ65_cP-zK1ft_image.png)

- ****<span style={{color:'#2166ae'}}>&lt;%JSONin%&gt;</span> will contain the JSon data, the default will use. The variable will be of Type Json, Single Value.

```json
{
	"store": {
		"book": [
			{
				"category": "reference",
				"author": "Peter Mark Roget",
				"title": "Rogets 21st Century Thesaurus",

			},
			{
				"category": "fiction",

				"title": "The Hare",
				"price": 12.89
			},
			{
				"category": "fiction",
				"author": "Kristin Hannah",
				"title": "The Four Winds",
				"isbn": "0-573-21321-3",
				"price": 8.69
			},
			{
				"category": "fiction",
				"author": "Lauren Oyler",
				"title": "Fake Accounts",
				"isbn": "0-698-19225-3",
				"price": 21.39
			}
		],
		"bicycle": {
			"color": "blue",
			"price": 19.65
		}
	}
}
```

- ****<span style={{color:'#2166ae'}}><span style={{color:'#2166ae'}}>&lt;%Filter%&gt;</span></span> will contain the JQuery expression that we will use in order to extract from the Json input data. The variable will be of Type String and Single Value. The query will retrieve the last book in order (notice that the result of the query will be a list with one value). This is the filter for obtaining the last book: `$..book[-1:]`

  <span style={{color:'#2166ae'}}>&lt;%result%&gt;</span> will be our actions output. The variable will be a List of type Json.

**Further queries**

We will continue with further examples of queries by only changing the <span style={{color:'#2166ae'}}>&lt;%Filter%&gt;</span> and showcasing the returned results for the same Json input, below the examples.

- <span style={{color:'#2166ae'}}>&lt;%Filter%&gt;</span> will use `$..book[0,1]` to obtain the first two books:

![](/images/docs/JEFDMyXt4IQb59smu1L87_image.png)



- <span style={{color:'#2166ae'}}>&lt;%Filter%&gt;</span> will use `$..book[?(@.isbn)]` to filter all books with isbn number:

![](/images/docs/0vJCpPjuuqdXPc5OjtGdI_image.png)

- <span style={{color:'#2166ae'}}>&lt;%Filter%&gt;</span> will use `$..book[?(@.price<10)]` to filter all books cheaper than 10:

![](/images/docs/jRPoNGo2ODM_ylXZFOFcT_image.png)
