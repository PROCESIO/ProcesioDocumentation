Throws a custom exception when this action is reached.

![](/images/docs/zfhz6LLDOomaWDnu6euXW_image.png)

## Inputs & Outputs

- **\[ I\* ] Custom Message:**  representing the error message.

## Example Usage

- Drag action onto the canvas to get familiarized with it.

![](/images/docs/C0MBb58GgHYkVIYswhn0s_tmp.png)

- **Save & Run** the process and then **Check Instance** to see the output.

![](/images/docs/enGzhVR6IJJ-GXIEeU7nJ_tmp.png)

## Use Cases

- Throwing custom errors and setting the process status as **failed** when a specific condition is met.
