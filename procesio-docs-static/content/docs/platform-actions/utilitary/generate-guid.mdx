Use **Generate Guid** action to generate GUIDs. You can use it as a unique reference number, when needed to identify information like ID hardware, software, accounts, documents and other items.



### How to configure the Generate Guid action?

1\.   Create a process and give it a name.

2\.    Drag the **Generate Guid&#x20;**&#x61;ction to the canvas and link it to the other actions.

![](/images/docs/_EsB7O4NzHJOKvP-Y89em_screenshot-2022-08-25-at-162328.png)

3\. Create the variables needed for the configuration of the action, and then add them to the configuration panel:

![](/images/docs/bXcbqOb5xn591aSDCbbgK_screenshot-2022-08-25-at-161857.png)

4\. Once you run the process, you can check in instances the value of the generated guid in Output:&#x20;

![](/images/docs/5lh7QKMcCT3bNrAfrd1V1_screenshot-2022-08-25-at-162534.png)
