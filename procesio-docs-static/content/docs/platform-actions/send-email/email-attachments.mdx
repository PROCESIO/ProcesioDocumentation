Email attachments is a must have when building automations that involve email communications.

### **Prerequisites**

1. Procesio [Document Template](/docs/overview/processes/document-designer/document-template-creation)
2. SMTP server

### **How to**

&#x20; 1\.    Create the [SMTP credential](/docs/overview/processes/untitled/working-with-credentials).

&#x20; 2\.    Create the [document template](/docs/overview/processes/document-designer/document-template-creation) (you have the ability to use variables in your templates).

![](/images/docs/Nh3Rx81AKTJeKOpbN2VBb_image.png)

&#x20; 4\.    Create the [process](/docs/overview/processes/process-designer/your-first-process).

![](/images/docs/bxUyLtq-qjUPdXn4LUtmh_image.png)

&#x20; 5\.    Configure the actions:

a. [Generate document](/docs/platform-actions/call-api/generate-document)

Create the process variables:

- <span style={{color:'#2166ae'}}>&lt;%vacation%&gt;</span> - output variable of type File.
- <span style={{color:'#2166ae'}}>&lt;%StartDate%&gt;</span> - input DateTime variable.
- <span style={{color:'#2166ae'}}>&lt;%EndDate%&gt;</span> - input DateTime variable.
- <span style={{color:'#2166ae'}}>&lt;%Reason%&gt;</span> - input String variable.
- <span style={{color:'#2166ae'}}>&lt;%Name%&gt;</span> - input String variable.

Click the **Generate Document** action:

- Select the document template\*
- type a name for the file you will create
- choose the File variable that will store the document.

![](/images/docs/UrAT5-d-YRdqzV_pb7JaV_image.png)

Click **Map Document Data** and use the following configuration

![](/images/docs/-LNzp4xNM71GfqusbVsHV_image.png)

b. **List Add**

- create the list variable <span style={{color:'#2166ae'}}>&lt;%attachment%&gt;</span> of type File.
- use the following configuration to add the <span style={{color:'#2166ae'}}>&lt;%vacation%&gt;</span> file variable in to the <span style={{color:'#2166ae'}}>&lt;%attachment%&gt;</span> variable.

![](/images/docs/x4esHAZJVnvE8jX7oSGnO_image.png)

c. [Send Email](/docs/platform-actions/call-api/send-email) - add the configuration and map the <span style={{color:'#2166ae'}}>&lt;%attachment%&gt;</span> variable to the email.

![](/images/docs/rYBQZwjATrPfvnXdXq6Hs_image.png)



&#x20; 6\.    **Save**, **Validate&#x20;**&#x61;nd **Run&#x20;**&#x74;he process. You will be asked to enter the values of the input values. Click **Run&#x20;**&#x61;gain.

![](/images/docs/r8Kca-ZXxVM2Hcsjcfmwv_image.png)

&#x20; 7\.    Click **Check instance** to view the result.
