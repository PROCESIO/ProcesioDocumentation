This action extracts the month from a <span style={{color:'#ff9307'}}>DateTime</span>.

![](/images/docs/jvsHRtG3lOYJRAiqi5lCC_tdy.png)

## Inputs & Outputs

- **\[ I ] Date**: <span style={{color:'#ff9307'}}>DateTime</span> to extract month from.
- **\[ O ] Month**: <span style={{color:'#0039e3'}}>Integer</span> representing the extracted month.

![](/images/docs/If95wtobdFgGS2xyoFgKo_tdy.png)

## Example Usage

- Drag action onto canvas to get familiarized with it.

![](/images/docs/nlmJnROmusrU50uwRcMjx_tdy.png)

- Declare some variables to be used with the action.

![](/images/docs/KlINbioHbjzQ_9hA87K7O_tdy.png)

- Assign the newly created variables.
- **Save & Run** the process and then **Check Instance** to see the output.

![](/images/docs/Ibt4f5DnuJ1j_9N2eRySN_tdy.png)
