This action extracts the year from a <span style={{color:'#ff9307'}}>DateTime</span>.

![](/images/docs/zSia8DuW2l9-MUUip0k49_tdy.png)

## Inputs & Outputs

- **\[ I ] Date**: <span style={{color:'#ff9307'}}>DateTime</span> to extract year from.
- **\[ O ] Year**: <span style={{color:'#0039e3'}}>Integer</span> representing the extracted year.

![](/images/docs/DPyLojlVoqeup16jrVBuJ_tdy.png)

## Example Usage

- Drag action onto canvas to get familiarized with it.

![](/images/docs/vxs7ZlGYHJxjcLN1nizMY_tdy.png)

- Declare some variables to be used with the action.

![](/images/docs/rygVEb7aZgmbfCmkpYdmA_tdy.png)

- Assign the newly created variables.
- **Save & Run** the process and then **Check Instance** to see the output.

![](/images/docs/LmmnxjkOnFD6bft6Md80c_tdy.png)
