This action transforms a **<span style={{color:'#ff9307'}}>DateTime</span>** to <span style={{color:'#1c417e'}}>String</span>, using a default format specification.

![](/images/docs/qywG7plPbqj2JOj_XrVl3_tdy.png)

## Inputs & Outputs

- **\[ I ] Datetime**: DateTimeto be transformed.
- **\[ I ] Language**: Language to be used for month names and days of the week when formatting.
  - <span style={{color:'#00ced1'}}>Drop-down</span> with **multiple languages**.
- **\[ I ] Format**: Formatting to be used when transforming.
  - <span style={{color:'#00ced1'}}>Drop-down</span> with a few **common date formats**.
- **\[ O ] Formatted datetime**: <span style={{color:'#1c417e'}}>String</span> representing the formatted date.

![](/images/docs/xfvlGeMfiPUg4IzTXu9Xg_tdy.png)

- Formatting <span style={{color:'#ff9307'}}>DateTime</span> values into a more human readable format.
