This action checks if a date is on a weekday.

![](/images/docs/hhtqRsNWW_s88GaZcnE-L_image.png)

## Inputs & Outputs

- **\[ I ]** **Date**: <span style={{color:'#ff9307'}}>DateTime</span> representing the date to be checked.
- **\[ O ] Is it weekday ?**: <span style={{color:'#7239e3'}}>Boolean</span> representing the logic value of `Is Date on a weekday ?`

![](/images/docs/KCdJPPkmLK6evHYWZ-nDo_tdy.png)

## Example Usage

- Drag action onto canvas to get familiarized with it.

![](/images/docs/rshotUJ7BWaibRhsrtbEV_tdy.png)

- Declare some variables to be used with the action.

![](/images/docs/zKDLBUSekLBA0dSKj3Vcp_tdy.png)

- **Save & Run** the process and then **Check Instance** to see the output.

![](/images/docs/M9Hz6xts4NMmyo_7QNAv5_tdy.png)

## Use Cases

- Checking if a date is on a weekday or not.

## Tips

- To check if a date is on a weekend, consider using [Is Weekend](/docs/platform-actions/call-api/is-weekend).
