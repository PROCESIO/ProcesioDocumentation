This action gets the current date.

![](/images/docs/Q005optkPqpoVraheabD__image.png)

## Inputs & Outputs

- **\[ O ]** **Current date**: <span style={{color:'#ff9307'}}>DateTime</span> containing the current date only. &#x20;

![](/images/docs/Ao9WSgAZ8L7avCSF00pEu_tdy.png)

## Example Usage

- Drag action onto canvas to get familiarized with it.

![](/images/docs/38KCJPCfmePinxkuw1MAD_tdy.png)

- Declare some variables to be used with the action.

![](/images/docs/iPCn3TR4WoTXXUR7XSvY2_tdy.png)

- Assign the newly created variables.
- **Save & Run** the process and then **Check Instance** to see the output.



![](/images/docs/-ZVTjluMkVwd2pywP4nfd_tdy.png)

## Use Cases

- Mentioning the current date in documents created using the [Document Designer](/docs/overview/processes/document-designer/document-template-creation).

## Tips

- The current date is in UTC standard so timezone differences should be considered.
- If you also need the time and not just the date, consider using [UTC Now](/docs/platform-actions/call-api/utc-now).
