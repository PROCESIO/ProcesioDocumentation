&#x20; This action transform a <span style={{color:'#ff9307'}}>DateTime</span> to <span style={{color:'#1c417e'}}>String</span>, using a custom format specification.

![](/images/docs/2dWHH022kJdZQikkfM2HR_tdy.png)

## Inputs & Outputs

- **\[ I ] Datetime:** <span style={{color:'#ff9307'}}>DateTime</span> to be transformed.
- **\[ I ] Language**: Language to be used for month names and days of the week when formatting.
  - <span style={{color:'#00ced1'}}>Drop-down</span> with **multiple languages.**
- **\[ I ] Format**: <span style={{color:'#1c417e'}}>String</span> representing the wanted format.
- **\[O] Formatted datetime**: <span style={{color:'#1c417e'}}>String</span> representing the formatted date.

![](/images/docs/BQLI_tTHTZV2H0NOpe20l_tdy.png)

## Use Cases

- Formatting <span style={{color:'#ff9307'}}>DateTime</span> values into a more human readable format.
- Translating month names or days of the week from one language to another using the [Parse <span style={{color:'#ff9307'}}>DateTime</span>](/docs/platform-actions/call-api/parse-datetime) and **Format <span style={{color:'#ff9307'}}>DateTime</span>** actions together.

## Tips

- The **Format** field uses standard .NET format specifiers which are documented [here](https://docs.microsoft.com/en-us/dotnet/standard/base-types/custom-date-and-time-format-strings).
- The **Language** selector picks which language should be used when the wanted format contains :
  - **Month names**: June (UK) 🆚 Iunie (RO) 🆚 Juin (FR)
  - **Days of the week**: Monday (UK) 🆚 Luni (RO) 🆚 Lundi (FR)
  - **Abbreviations**&#x20;
