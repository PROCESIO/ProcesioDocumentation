This action extracts the hour from a <span style={{color:'#ff9307'}}>DateTime</span>.

![](/images/docs/plvMynxvUIXBwieOvWTks_tdy.png)

## Inputs & Outputs

- **\[ I ] Datetime**: <span style={{color:'#ff9307'}}>DateTime</span> to extract hour from.
- **\[ O ] Hour**: <span style={{color:'#0039e3'}}>Integer</span> representing the extracted hour.

![](/images/docs/wsf3yvzg2eICBHneLdo9O_tdy.png)

## Example Usage

- Drag action onto canvas to get familiarized with it.

![](/images/docs/Hjf-hGBTAA7sFIq08I22s_tdy.png)

- Declare some variables to be used with the action.

![](/images/docs/OAOnU7rvHuYq9s2wz3Pfp_tdy.png)

- Assign the newly created variables.
- **Save & Run** the process and then **Check Instance** to see the output.

![](/images/docs/3p8DYkOArEZQ12BGctPSi_tdy.png)
