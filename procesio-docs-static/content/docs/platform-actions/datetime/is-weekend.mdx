This action checks if a date is on a weekend.

![](/images/docs/kwfqoz6L0Oo3RWjCjQXC0_image.png)

## Inputs & Outputs

- **\[ I ]** **Date**: <span style={{color:'#ff9307'}}>DateTime</span> representing the date to be checked.
- **\[ O ] Is it weekend ?**: <span style={{color:'#7239e3'}}>Boolean</span> representing the logic value of `Is Date on a weekend ?`

![](/images/docs/UKZLgORPNKytX5GMBZghs_tdy.png)

## Use Cases

- Checking if a date is on a weekend or not.

## Tips

To check if a date is on a weekday, consider using [Is Weekday](/docs/platform-actions/call-api/is-weekday).
