This action computes the difference between two **<span style={{color:'#ff9307'}}>DateTime</span>** values. **\[hour, minute, second]**

![](/images/docs/RawnYz2DwRIXXnND7AQoI_tdy.png)

## Inputs & Outputs

- **\[ I ] Unit of time**: <span style={{color:'#00ced1'}}>Drop-down</span> to choose which unit of time will be used for measuring the difference.
  - **Hour**
  - **Minute**
  - **Second**
- **\[ I ] First Datetime**: <span style={{color:'#ff9307'}}>DateTime</span> representing the minuend.
- **\[ I ] Second Datetime**: <span style={{color:'#ff9307'}}>DateTime</span> representing the subtrahend.
- **\[ O ] Difference**: <span style={{color:'#0039e3'}}>Integer</span> equal to the result of **First Datetime - Second Datetime** in **Unit of time**.

![](/images/docs/jFmRi56eIgu52MNqvOqXQ_tdy.png)

## Use Cases

- Computing the time difference between two <span style={{color:'#ff9307'}}>DateTime</span> values.

## Tips

- If **Second Datetime** is lesser than **First Datetime**, the **Difference** will be negative.
- To compute date difference, consider using [Date Difference](/docs/platform-actions/call-api/date-difference).
