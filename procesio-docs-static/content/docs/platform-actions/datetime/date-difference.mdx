This action computes the difference between two **DateTime** values. **\[year, quarter, month, week, day]**

![](/images/docs/jeFjYaz26_CPub8asp0iK_tdy.png)

## Inputs & Outputs

- **\[ I ] Unit of time**: Drop-down to choose which unit of time will be used for measuring the difference.
  - **Year**
  - **Quarter**
  - **Month**
  - **Week**
  - **Day**
- **\[ I ] First Datetime**:**&#x20;&#x20;**DateTime representing the minuend.&#x20;
- **\[ I ] Second Datetime**: DateTime representing the subtrahend.
- **\[ O ] Difference:&#x20;**Integer equal to the result o&#x66;**&#x20;First Datetime - Second Datetime&#x20;**&#x69;&#x6E;**&#x20;Unit of time**.

![](/images/docs/RHC4iYi1tqtbiyfQLeAUd_tdy.png)

## Use Cases

- Computing the date difference between two DateTime values.

## Tips

- If **Second Datetime** is lesser than **First Datetime**, the **Difference** will be negative.
- To compute time difference, consider using [Time Difference](https://docs.procesio.com/how-to/time-difference).
