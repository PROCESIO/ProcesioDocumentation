This action computes the difference between two **<span style={{color:'#ff9307'}}>DateTime</span>** values. **\[year, quarter, month, week, day]**

![](/images/docs/jeFjYaz26_CPub8asp0iK_tdy.png)

## Inputs & Outputs

- **\[ I ] Unit of time**: <span style={{color:'#00ced1'}}>Drop-down</span> to choose which unit of time will be used for measuring the difference.
  - **Year**
  - **Quarter**
  - **Month**
  - **Week**
  - **Day**
- **\[ I ] First Datetime**:**&#x20;&#x20;**<span style={{color:'#ff9307'}}>DateTime</span> representing the minuend.&#x20;
- **\[ I ] Second Datetime**: <span style={{color:'#ff9307'}}>DateTime</span> representing the subtrahend.
- **\[ O ] Difference:&#x20;**<span style={{color:'#1c417e'}}>Integer</span> equal to the result o&#x66;**&#x20;First Datetime - Second Datetime&#x20;**&#x69;&#x6E;**&#x20;Unit of time**.

![](/images/docs/RHC4iYi1tqtbiyfQLeAUd_tdy.png)

## Use Cases

- Computing the date difference between two <span style={{color:'#ff9307'}}>DateTime</span> values.

## Tips

- If **Second Datetime** is lesser than **First Datetime**, the **Difference** will be negative.
- To compute time difference, consider using [Time Difference](/docs/platform-actions/call-api/time-difference).
