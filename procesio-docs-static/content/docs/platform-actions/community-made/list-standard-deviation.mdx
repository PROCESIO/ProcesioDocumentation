This action computes the standard deviation, which measures the amount of variation or dispersation in the entire population of data points.

![](/images/docs/Z3xZ_6D21xIMvvU4_1n5D_image.png)

## Inputs & Outputs

- **\[ I ] List of numbers**: <span style={{color:'#368b10'}}>Double (List)</span> representing data points for calculating the standard deviation.
- **\[ O ] Standard deviation**: <span style={{color:'#368b10'}}>Double (Single Value)</span> representing the measure of variability in the data set derived from the input list.

![](/images/docs/3BLHE88mRryzSkAzZUS5J_image.png)

## Use Cases

- By analyzing the population standard deviation for page load times and response times, you can identify patterns that may cause slow or inconsistent user experiences, pinpointing potential performance bottlenecks.

## Tips

- When working with data representing an entire population (full population data), opt for the [List Standard Deviation](/docs/platform-actions/call-api/list-standard-deviation) action to accurately measure the variability. However, if you are dealing with a sample from a larger population, do NOT use the List Standard Deviation action.
