This action computes the difference between two simple lists.

![](/images/docs/TWDzOnGrV1thKkvMJ4scX_tmp.png)

## Inputs & Outputs

- **\[ I ] First list:** **List\&lt;**<span style={{color:'#be241f'}}>Object</span>**>** representing the first list (**A**)
- **\[ I ] Second list:** **List\&lt;**<span style={{color:'#be241f'}}>Object</span>**>** representing the second list (**B**)
- **\[O] Difference**: **List\&lt;**<span style={{color:'#be241f'}}>Object</span>**>** representing the difference (**A-B**)

![](/images/docs/BxuABmFcOvaMUawPTOd_f_tmp.png)
