This action computes the intersection between two simple lists.

![](/images/docs/KOEr70n4XpDdXw_RKGKPn_tmp.png)

## Inputs & Outputs

- **\[ I ] First list:** **List\&lt;**<span style={{color:'#be241f'}}>Object</span>**>** representing the first list (A)
- **\[ I ] Second list:** **List\&lt;**<span style={{color:'#be241f'}}>Object</span>**>** representing the second list (B)
- **\[O] Intersection**: **List\&lt;**<span style={{color:'#be241f'}}>Object</span>**>** representing the intersection (&#x41;**∩**&#x42;)
