### Overview

Use this action to delete a sheet by name from an existing workbook.

### How to configure the Delete Sheet action?

Step 1. Create a process and give it a name.

Step 2. Drag the **Delete Sheet** action to the canvas and link it to the other actions.

![](/images/docs/KVjdxJI82kKpMdO8yAZDO_image.png)

Step 3. Create the variables needed for the configuration of the action, and then add them to the configuration panel:

![](/images/docs/J_xrbpMjpYGEWCj4w8lw-_screenshot-2022-09-22-at-125249.png)

- **Excel workbook**: you need to input the file with the workbook where you want to delete the sheet
- **Sheet name**: you need to input a string with the sheet name which you want to delete

Step 4. **Save**, **Validate&#x20;**&#x61;nd **Run&#x20;**&#x74;he process.

Step 5. You will be prompted to add the file for the input variable. Click **Run**.

Step 6. To verify the process click **Check instance**.

Step 7. Open the file; it will no longer contain the deleted sheet.
