## Overview

The `XLS to XLSX` action converts an XLS file to an XLSX file.&#x20;

<Callout type="info">
Styles will not be ported; the maximum permitted size for the XLS file is 10 Mb.
</Callout>

![](/images/docs/b7LC0zqqLZjyRabkpNUkJ_xls-toxlsx.png)

### Parameters

### Inputs

- **Source XLX&#x20;**&#x20;(`file`). The input XLX file.

### Outputs

- **Destination XLSX** (`file`): The output XLSX file.
