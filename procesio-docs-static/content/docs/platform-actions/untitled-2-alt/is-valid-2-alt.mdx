The **Is Valid&#x20;**&#x61;ction verifies if a PDF is valid.

![](/images/docs/kQYGkEL05y7kn0P4BQlHz_image.png)

### How to configure the Is Valid action?

&#x20;    1\. Create a process and give it a name.

&#x20;    2\. Drag the **Is Valid** action to the canvas and link it to the other actions.

&#x20;    3\. Create the variables needed for the configuration of the action, and then add them to the configuration panel:&#x20;

### Inputs & Outputs

- **\[ I ]** **Pdf file**: Filerepresenting the input pdf file.
- **\[ O ] Validation output**: Boolean representing if the pdf is valid or not.
- **\[ O ] Validation reason:** String representing the error if the validation failed.

![](/images/docs/MVadv1nuwfx6ZDE3EYtrz_image.png)

&#x20;    4\. **Save**, **Validate** and **Run** the process.

&#x20;     5\. You will be prompted to add the file for the input variable. **Drag** or **upload&#x20;**&#x74;he file and click **Run**.

&#x20;    6\. To verify the process click **Check Instances**.

&#x20;    7\. In the output section you will see if the pdf is valid and if not also the validation reason.
