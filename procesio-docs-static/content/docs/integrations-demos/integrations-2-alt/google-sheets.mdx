You can use **PROCESIO** to create an automated workflow that updates your inventory every time a new order is placed. And by integrating **PROCESIO** with **Google Sheets**, you can have all that data automatically transferred and updated in a spreadsheet that you can access from anywhere, at any time.

🤖 And the best part? You don't need to be a tech whiz to integrate **PROCESIO** with **Google Sheets**. The process is straightforward and easy to follow, and you can find lots of resources online to guide you.

## Obtaining the OAuth Authentication Token

To generate an OAUTH token, we'll need 3 things:

- client_id - part of the Google Sheets API Credential we just created.
- client_secret - also part of the Google Sheets API Credential we just created.
- code - secret code valid for 60 seconds after obtaining it.

### **Finding&#x20;**client_id**&#x20;and&#x20;**client_secret

- From within [Google API Console](https://console.cloud.google.com/) navigate to the credential tab, and click the credential you've just created.&#x20;

<Callout type="warn">
Make sure you have the right project selected.
</Callout>

![](/images/docs/lkO8yKRgdwnvlhd-X0gVI_image.png)

- Now copy the values for **Client ID** and **Client secret** and save them for later.

![](/images/docs/legmblPu8duSmmkU0OqHP_image.png)

### **Preparing the request**

Now that we have the **client\_id** and **client\_secret** ,we only need to get the code so that we can use it to receive the **OAuth token**. The code has a 60 second validity so before obtaining it, we'll prepare a request in **Postman** so that we can just **Obtain the code** & **Send the request** straight away.

- Open **Postman** and configure a **POST** request with **x-www-form-urlencoded** body type.

**POST:** [https://accounts.google.com/o/oauth2/token](https://accounts.google.com/o/oauth2/token)

- Set the following key-value pairs
  - **grant\_type&#x20;**➜ `authorization_code`
  - code ➜ here you need to add the code we obtain in the next step
  - client_id ➜ Value of **Client** **ID** that you saved.
  - client_secret ➜ Value of **Client secret** that you saved.
  - **redirect\_uri** ➜ [`https://procesio.com/`](https://procesio.com/)
    **access\_type&#x20;**➜ `offline`

![](/images/docs/Repj2eOUMOz2_aPgixc4O_image.png)

### **Getting the&#x20;**code

The request is ready to be sent and only needs the code. To obtain the code you need to:

- Build an URL as per below:

[https://accounts.google.com/o/oauth2/auth?access\_type=offline\&prompt=consent\&client\_id=\&response\_type=code\&scope=https://www.googleapis.com/auth/spreadsheets\&redirect\_uri=https://procesio.com/](https://accounts.google.com/o/oauth2/auth?access_type=offline\&prompt=consent\&client_id=YORCIENTID.apps.googleusercontent.com\&response_type=code\&scope=https://www.googleapis.com/auth/spreadsheets\&redirect_uri=https://procesio.com/)

In the green highlighted section you will insert the value of Client ID just like in the request.&#x20;

- Copy the URL & paste it into a browser. This should prompt you to select a corresponding account to which resources will become accessible via OAuth.

<Callout type="warn">
Make sure to sellect the right account.
</Callout>

![](/images/docs/LoT-yIM9YUwHxCjKIKXgu_image.png)

- Once you logged in, press the **Allow** button

![](/images/docs/uOD6fuTLebJ0E6K-nSXUD_image.png)

- After allowing, you’ll be redirected to a screen that should look something like below. This new URL (on which you were redirected) contains the code to be used in POSTMAN for obtaining the Access Token as well as the Refresh Token. (We have about 60 s until the code expires)

![](/images/docs/GegVV7nc5RQlJJCIztYLX_image.png)

- Complete your **Postman** request by adding the **code** and hit **Send**.



![](/images/docs/iRLIYrD3JsPANpRLZhRvw_image.png)

- You have now obtained the access token and the refresh token. Make sure to save them 💪

<Callout type="error">
The access token expires in **one hour** so a refresh mechanism needs to be put in place.

More on how to refresh the token [here](https://docs.informatica.com/integration-cloud/cloud-data-integration-connectors/current-version/google-sheets-connector/introduction-to-google-sheets-connector/administration-of-google-sheets-connector/generating-oauth-2-0-access-tokens.html).
</Callout>

## Using PROCESIO with Google Sheets (Append Demo)

- Build a basic process with **Start** ➜ **Call API** ➜ **Stop**
- Create a Rest API Credential with the base URL: [https://sheets.googleapis.com/v4/spreadsheets](https://sheets.googleapis.com/v4/spreadsheets)

<Callout type="success">
We are going to skip testing the credential.
</Callout>

![](/images/docs/GK2znq_XGpUHFhuapYwTW_image.png)

- Create some variables in your process

| Name                    | Type                                     | Default value                                      | Single value | Set as     |
| ----------------------- | ---------------------------------------- | -------------------------------------------------- | ------------ | ---------- |
| **Token**               | String      | `Bearer `access_token | ✔            | -          |
| **SheetsAPIStatus**     | ****Integer |                                                    | ✔            | **Output** |
| **SheetsAPIBodyOutput** | Json        |                                                    | ✔            | **Output** |



![](/images/docs/0AFkQn-5AzEoGSaleppAX_image.png)

- Configure the **Call API** Action as follows:

**Verb&#x20;**➜ `POST`

**MethodName&#x20;**➜ `/spreadsheets/{SheetID}/values/A1:append`

**Query Params**:

| Key              | Value         |
| ---------------- | ------------- |
| valueInputOption | USER\_ENTERED |

**Headers**:

| Key           | Value   |
| ------------- | ------- |
| Authorization | `Token` |

**Body**:

```json

{
    "range": "A1",
    "majorDimension": "ROWS",
    "values": [
        [
            "Sample Value"
        ]
    ]
}
```

![](/images/docs/gEqelDnai3tytBZLeXTA5_image.png)

- **Run the Process & Check your spreadsheet.**

<Callout type="info">
More details on the Google Sheets API can be found [here](https://developers.google.com/sheets/api/reference/rest/v4/spreadsheets.values/append).
</Callout>
