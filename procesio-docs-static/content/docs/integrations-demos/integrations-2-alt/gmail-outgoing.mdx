Sometimes you will need to send an email from a printer, scanner, or app using an SMTP relay service. Google provides such a service for free and you can easily add it to your flow to easily increase functionality.

### **Google account**

&#x20; 1\.    From the **Google Apps** menu, navigate to your Google Account.

![](/images/docs/4EA1o0pwR7tjXFMYxYQjH_image.png)

&#x20; 2\.    From the **Security** tab navigate to Signing in to Google and turn on the **2-Step Verification**.

&#x20; 3\.    From the Security tab navigate to **Signing in to Google** and select **App passwords**.

&#x20; 4\.    Select **Mail** and **Other(Custom name)** (for our case) and provide a name for this password. You will be able to see the generated password only once.

![](/images/docs/dDkQvgi5G04579iyU-umb_image.png)

### **Create the credential**

&#x20; 1\.    Navigate to the [Credentials Manager](https://docs.procesio.com/working-with-credentials) section.

&#x20; 2\.    Select the **Add New** button from the upper right corner.

&#x20; 3\.    Give a Name to the newly created connection, select SMTP Server Configuration from the **Credential Type&#xA0;**&#x64;ropdown and optionally add the Description.



![](/images/docs/-kA4aDXld2tcUD837BIsy_image.png)



&#x20; 4\.    Select th&#x65;**&#xA0;Next step** button.

&#x20; 5\.    Enter the GMail connection data as shown in the screenshot. The password is the one that was generated in a previous step.

![](/images/docs/5RT63MxwZMRGSRv3g3uKz_image.png)

&#x20; 6\.    When you click the **Test connection** button PROCESIO will test the connection and give the Successfully tested! message.

&#x20; 7\.    **Save** the connection.

### **Create and configure the process**

&#x20; 1\.    After creating the flow using the drag and drop functionality you can give the Send Email Action a name, description and from the Select SMTP Server dropdown select the connection created at step 7.

![](/images/docs/4pcQx4h5MsBoEZqNP10kt_image.png)

&#x20; 2\.    In the **Send Email** screen enter the **From**(display name), **To**, Cc, Bcc, **Subject** and **Body** information and click **Save**.

&#x20; 3\.    If the Body will use HTML tags you will need to check **Body is HTML**.

![](/images/docs/xX5CCXY8mVuiNO4PbAV4N_image.png)

&#x20; 4\.    From the upper right corner you can **Save**, **Validate** and **Run** the Process.

![](/images/docs/iOvMMG04hOxVvljPqUWS3_image.png)
