You can open and read your messages from Gmail in PROCESIO as it supports POP.

## 1. Set up POP in Gmail

1. On your computer, open [Gmail](https://mail.google.com/).&#x20;
2. In the top right, click **Settings > See all settings**
   - Click the **Forwarding and POP/IMAP** tab.
   - In the "POP download" section, select **Enable POP for all mail** or **Enable POP for mail that arrives from now on**.
   - At the bottom of the page, click **Save Changes**.

## 2. Next, create an inbound email credential in PROCESIO

Go to **PROCESIO > Credentials Manager.&#x20;**

Clic&#x6B;**&#x20;"+Create new".**

Give it a name and select credential type "**SMTP Inbound Configuration**".

| Incoming Mail (POP) Server name | pop.gmail.com                                                                                                                                                                  |
| ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Protocol type                   | POP3                                                                                                                                                                           |
| Port number                     | 995                                                                                                                                                                            |
| Encryption                      | Auto                                                                                                                                                                           |
| Username                        | Your email address                                                                                                                                                             |
| Password                        | Your **App Password** <span style={{color:'#ff6900'}}>(</span><span style={{color:'#e71616'}}>not the same as your Gmail password)</span>; read the below steps on how to generate an app password |



## 3. Go to your Google account and generate an app-specific password

<Callout type="info">
**Sign in with App Passwords**

**Tip:&#xA0;**&#x41;n App Password is a 16-digit passcode that gives PROCESIO permission to access your Google Account. App Passwords can only be used with accounts that have [2-Step Verification](https://support.google.com/accounts/answer/185839) turned on.
</Callout>

### Create & use App Passwords

If you use [2-Step-Verification](https://support.google.com/accounts/answer/185839) and get a "password incorrect" error when you sign in, you can try to use an App Password.

1. Go to your [Google Account](https://myaccount.google.com/).
2. Select **Security**.
3. Under "Signing in to Google," select "**2-Step Verification**". You may need to sign in. If you don’t have this option, it might be because:
   1. 2-Step Verification is not set up for your account.
   2. 2-Step Verification is only set up for security keys.
   3. Your account is through work, school, or other organization.
   4. You turned on Advanced Protection.
4. At the bottom of the page, select "**App passwords"&#x20;**&#x61;nd choose the app you using **Select device&#x20;**&#x61;nd choose the device you’re using **Generate**. If you can't find it, here is a link: [https://myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)
5. Follow the instructions to enter the App Password. The App Password is the 16-character code in the yellow bar on your device.
6. Tap **Done**.

<Callout type="info">
Use the generated app password to create/update the credential for Inbound Configuration in PROCESIO (in step 2).
</Callout>
