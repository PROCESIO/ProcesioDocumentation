**Scenario:** We will create a process flow that notifies decisional factors, if the products contained in an excel sheet are within budget range.

This will be a basic scenario where PROCESIO can help you with sending criteria based alerts to management, notifying them on actions that need to be taken.

This particular flow has the aim of sending an SMS alert as well as an e-mail to decisional factors, for each item priced above or equal to 100 $ (these items are considered to be above budget). For items priced below 100$ (items within budge range), only an e-mail alert will be sent to decisional factors. In this case, the email will state that the item is within the budget range, and therefore order placement can take place.

Before you get started, you will need the following items in order to complete the suggested process:

- a configuration to an email server that will be used to send the email. ([see here](https://docs.procesio.com/working-with-credentials))
- a configuration to a REST API of an web-to-sms service. ([see here](https://docs.procesio.com/rest-api-credential)) 
  - the above require creation of 2 connection configurations in the Credentials Manager.
- Excel/ CSV file containing tabular data

## **Let's build this process!&#xA0;**

The process flow created will look like this: 

![](/images/docs/eVeBqpFXOvivjP8t3g8HC_image.png)

### &#x20; Create and configure the process &#x20;

**Step 1**: Drag and drop **Get File Data** into the canvas connect it to the **Start** Button.

Drag and drop a **For Each** and connect it to the **Get File Data**. 

- Within the **For Each** border, drag and drop a **Decisional** action and connect it to the **Start** within the **For Each**
- Within the **For Each** border, drag and drop a **Call Api Action** action connect it to the **Decisional**.
  - Within th&#x65;**&#x20;For Each** border, drag and drop a **Send Email** action and connect it to the **Call Api Action**.
    - Within the **For Each** border, drag and drop a **Join** action and connect it to the **Send Email**
  - Within the **For Each** border, drag and drop a second **Send Email** and connected to the **Decisional** and to the **Join&#x20;**&#x61;ction.
- Connect the **For Each** border to the **Stop** action.

**Step 2:&#xA0;**&#x43;onfigure the **Get File Data** according to [documentation](https://docs.procesio.com/how-to/get-file-data). For configuring this step [video available here](https://www.youtube.com/watch?v=aLMgtMHQjb4).

**Step 3:&#xA0;**[Configure](https://docs.procesio.com/how-to/for-each) the **For Each**

- **In List -&#x20;**&#x77;e will use the variable/ data model, defined at Step 2.

![](/images/docs/Jyx7j123YoRsjhvQ20sFv_image.png)

- Set **Action timeout** to 200 milliseconds. In order to avoid an infinite loop, we want the "For Each" action to stop after 200 second of running (can be set in accordance to needs). 
- In the "For Each" output tab, you will mention the name of the item in the list used for each iteration. Refers to the line item in your excel file.

![](/images/docs/7Pox-GQE5B8MnF1n-pB_I_image.png)

**Step 4:&#xA0;**&#x43;onfigure the "Decisional" action within the "For Each" border. Each case of the [Decisional](https://docs.procesio.com/how-to/decisional) needs to be configured.

- For this demo, you will have to configure two cases:
  - greater than or equal to 100$
  - less than 100$

### Decisional logic

**Case 1**: we will use &lt;%CustomerCsvLine.price%&gt;, defined in the "For Each" output (mentioned in Step 3).

- We will set the &lt;%customercsvline.price%&gt; to be greater than or equal to 100.
  - Additionally at "Target", Go To, we will set "Call Api Action". This means that if the condition is met, the "Decisional" will follow the "Call Api Action" branch.

![](/images/docs/Z2Y-6Yx_nxLwcb1F65t7C_image.png)



**Case 2**: we will use &lt;%CustomerCsvLine.price%&gt;, defined in the "For Each" output (mentioned in Step 3).

- We will set the &lt;%customercsvline.price%&gt; to be less than 100.
  - Additionally at "Target", Go To, we will set "Send Email (less 100)". This means that if the condition is met, the "Decisional" will follow the "Send Email (less 100)" branch.

![](/images/docs/1MzfoZpY5-SrhfrzCwFCO_image.png)

**Step 5:&#xA0;**&#x43;onfigure "Call Api Action" & "Send Email (greater 100)" actions

- As mentioned at the beginning of this article, connections for "Call Api Action" as well as "Send Email" have to be created within "Credentials Manager". See in this [article](https://docs.procesio.com/working-with-credentials) how to set up connections in Credentials Manager for  "Call Api Action" as well as "Send Email".
- Once connections have been created in Credentials Manager:
  - "Call API Action":
    - Select Api configuration (previously configured in Credentials Manager) and proceed to configure request.

![](/images/docs/I8gAXD30qA7eP--q1cYhp_image.png)

- The request needs to be configured ("Configure Request" card) in accordance with the API provider documentation. In our case [SMSO API documentation](https://api-docs.smso.ro/#introduction).

![](/images/docs/i7igK3omA5mr_8_MsaDsG_image.png)

- In accordance with [SMSO API documentation](https://api-docs.smso.ro/#introduction), we will use the following Query Params:
  - To: &lt;%CustomerCsvLine.phonenumber%&gt; (leveraging the phone number contained in the excel file)
  - body: Can be customised according to needs
  - Sender: 4 (in accordance to SMSO API documentation)
- "Send Email (greater 100)":
  - Select SMTP Server (previously configured in Credentials Manager) and proceed to "Send Email" configuration.

![](/images/docs/AyhwThyjfwR0Ih9stksoa_image.png)

- Proceed by clicking on "Send Email" card and configure it.

![](/images/docs/J6GOZIoMH6YNZLMX-YUcD_image.png)

- **From:** Type a desired display name. (The e-mail addressed configured in the Credentials Manager will be leveraged).
- **To:** &lt;%CustomerCsvLine.email%&gt; (leveraging the e-mail address contained in the excel file).
- **Body:** Custom, according to your needs. 

<Callout type="info">
To open the variables selector in Body and add process variables to your message: press Insert key (for Windows) or Alt + Tab (for MacOS)
</Callout>

**Step 5:&#xA0;**&#xA0;Configure "Send Email (less 100)": 

- Select SMTP Server (previously configured in Credentials Manager) and proceed to Send Email configuration.

![](/images/docs/IqTEWUGTWM2cSdEpEOJq3_image.png)

- Proceed by clicking on **Send Email** card and configure it.

![](/images/docs/MWU7vyvY0VnB5mLHH8CXI_image.png)



- **From**: Type a desired display name.
- **To**: &lt;%CustomerCsvLine.email%&gt; (leveraging the e-mail address contained in the excel file)
- **Body**: Custom according to needs. Variables in the body of the e-mail can be inserted by pressing the **Insert&#x20;**&#x62;utton.

**Step 6:  Save&#x20;**& **Validate&#x20;**&#x50;rocess

**Step 7:**  **Run the&#x20;**&#x70;rocess.
