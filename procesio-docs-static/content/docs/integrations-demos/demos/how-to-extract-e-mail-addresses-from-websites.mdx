**Scenario:&#x20;**&#x46;rom different websites you will extract all email addresses using a third party Web-Scraping API tool. The extracted information will be moved to a spreadsheet software in this case “SmartSheet”


## Let's build this process!

In this scenario we have three elements:

1. Third party API webscraping tool: [https://app.scrapingbee.com/](https://app.scrapingbee.com/)&#x20;
2. Third party spreadsheet software: [https://app.smartsheet.com/](https://app.smartsheet.com/)
3. Postman
4. PROCESIO: [https://procesio.app/](https://procesio.app/)

## ScrapingBee

Using a free account to obtain your API key.

**Step 1.&#x20;**&#x47;o to [https://app.scrapingbee.com/](https://app.scrapingbee.com/), log into your account and from your dashboard make sure you are able to obtain your API key.&#x20;

This API key will be used later when creating an REST API credential in PROCESIO.

![](/images/docs/Wc2UrDi4Dx8uOKXVgpJP7_image.png__Obtaining_API_key_from_ScrapingBee_)

## SmartSheet

**Step 1.&#x20;**&#x47;o to [https:](https://app.smartsheet.com/), and log in. From the home screen click on create.



![](/images/docs/q7-OBT15sayEeqYkdiciY_image.png)

**Step 2.&#x20;**&#x53;elect the sheet type to be of type Grid, once selected name the Sheet and hit ok.

![](/images/docs/CHuGeF_wrRw1wbKdKQTbO_image.png__SmartSheet_Grid_Type_)

**Step 3.&#x20;**&#x49;n SmartSheet a sheet of type Grid will open. In this sheet we will only keep the first two columns. (Remove Columns 3-6, by selecting them, right click and hitting delete column)

![](/images/docs/DkVV7JwuuuUAns6RMMbPD_image.png)

**Step 4.&#x20;**&#x4F;btain your API key from SmartSheet by clicking Account -> Personal Settings -> API Access -> Generate new access token. Once API key generated, keep it at hand for later use.

![](/images/docs/GcgC0RoPbblKrk2bXrrlx_image.png)

## Postman

We will use Postman in order to obtain the column IDs of the SmartSheet, spreadsheet software in which we will write the extracted e-mail addresses from the websites.

**Step 1.&#x20;**&#x4F;pen Postman, by using [SmartSheet API documentation](https://smartsheet.redoc.ly/tag/crossSheetReferences#operation/list-crosssheet-references) we will add the following get request in Postman

End point: `https://api.smartsheet.com/2.0/sheets/{SheetId}`

Request method: `GET`

- Body: "\&#123;"version": 1&#125;"
- Body format: application/json
- Headers:

| Name            | Direction | Type   | Value                                                                         |
| --------------- | --------- | ------ | ----------------------------------------------------------------------------- |
| `Authorization` | In        | String | the Bearer token you kept at hand from SmartSheet Step 4,(eg: `Bearer *****`) |

> The Sheet ID can be obtained from SmartSheet by navigating to Browse -> Sheets -> Right click on the Sheet -> Properties

![](/images/docs/7C4TPD1nBWTh5rLCJB619_image.png__Sheet_Id_)

## PROCESIO (Create process that extracts e-mail addresses)

There are multiple actions involved in extracting e-mails from a website. The complete process should look like this:

![](/images/docs/zwZwEB7sp2MnAop5BuS_V_image.png__WebScrappin_Process_)

**Step 1.&#x20;**&#x46;rom within PROCESIO go to "Credential Manager", press the Add new button and use these values:

Name: `ScrapingBee`
Credential type: `REST API Configuration`

**Step 2.** Press the Next step button and use these values:
URL: [`https://app.scrapingbee.com`](https://app.scrapingbee.com)``
Method: `GET`
Test endpoint: `/api/v1`
Authentication method: `API key authentication`
Key: `api_key`

Value: `Your API Key obtained at ScrapingBee Step 1`

- [x] **Header**
- [x] **Quer**y para**meters**

**Step 3.&#x20;**&#x50;ress the Save button

**Step 4.** From  within PROCESIO, "Credentials Manager,  press the Add new button and use these values:

Name: `SmartSheet`
Credential type: `REST API Configuration`

URL: [`https://api.smartsheet.com`](https://api.smartsheet.com)``
Method: `GET`
Test endpoint:`/2.0/sheets`
Authentication method: `API key authentication`
Key: `Authorization`

Value: `Bearer Your API Key obtained at SmartSheet Step 4`

- [x] **Header**
- [x] **Query parameters**

<Callout type="info">
Please note that, for the SmartSheet API, the field Value should always contain Bearer, followed by a space before the actual value of the key.&#x20;
</Callout>

![](/images/docs/mAF6pNYh5cbCS-rYqzXPj_screenshot-2022-01-11-at-111948.png)



&#x20;**Step 5.&#x20;**&#x41;dd a ForEach Action, name it's node `ForEach site`.

Configure the Action by (select Action):

- Configure field **For Each Item**, by typing `Iterator` (this will automatically add a variable of type string to named `Iterator`to the field.
- Configure field **In List&#x20;**&#x62;y pressing add variable button-> Create New Variable, `URLList`. Once the variable is created select it to this field.

| Variable Name | Variable Type | List Value | Default Value                                                     | Set as |
| ------------- | ------------- | ---------- | ----------------------------------------------------------------- | ------ |
| URLList       | string        | ✔          | \["https\://www\.realitatea.net/contact","https\://procesio.com"] | Input  |

<Callout type="info">
The URL default value, contains the list of websites through which we, will iterate for extracting e-mail addresses.
</Callout>

- Configure field **Action timeout**, by typing 600 (this impies that after 600 seconds, the process will timeout)

![](/images/docs/eDKfRqq0EqGgm6UMxP-py_screenshot-2022-01-11-at-112410.png)


&#x20;**Step 6.&#x20;**&#x49;nside the ForEach frame previosly named "ForEach site", add a Call API Action and in the side panel set the Node name to `ScrapingBee`

**Step 7.** In Select API configuration select `ScrapingBee`

**Step 8.&#x20;**&#x50;ress the **Configure Request** button and add these values:

Verb: `Get`
Endpoint: `/api/v1`
Query Params:

| Key | Value      |
| --- | ---------- |
| url | `Iterator` |

Form Configure Request, for the **Status Output&#x20;**&#x66;ield, add a variable named `StsOut` of type integer, by clicking the **Add Variable&#x20;**&#x62;utton-> **Create New Variable.&#x20;**&#x4F;nce the variable is created select it to this field.

Form Configure Request, for the **Body Output&#x20;**&#x66;ield, add a variable named `HTMLObjectOut` of type object, by clicking the **Add Variable&#x20;**&#x62;utton-> **Create New Variable.&#x20;**&#x4F;nce the variable is created select it to this field.

| Variable Name   | Variable Type | Single Value | Set as |
| --------------- | ------------- | ------------ | ------ |
| `StsOut`        | integer       | ✔            | output |
| `HTMLObjectOut` | object        | ✔            | output |

![](/images/docs/iMfqwNJKKxe90lr2EOk1k_image.png)

<Callout type="info">
Please note that the key field is case sensitive and type url (in no caps) and in the value field add  the variable Iterator, from the + icon.
</Callout>


**Step 9.&#x20;**&#x49;nside the ForEach frame previously named "ForEach site" , add a **Map Process Data** Action.

Configure the Action by (select Action):

- Adding in the right input field, the variable `HTMLObjectOut`
- Adding in the left input field a new variable named `HTMLString` by hitting the **Add Variable&#x20;**&#x62;utton-> **Create New Variable.&#x20;**&#x4F;nce the variable is created select it to this field.

| Variable Name | Variable Type | Single Value | Set as |
| ------------- | ------------- | ------------ | ------ |
| `HTMLString`  | string        | ✔            | output |

![Map Process Data Configuration](/images/docs/5bCw4YHO5tdzbA75tQGPF_image.png)

**Step 10.&#x20;**&#x49;nside the ForEach frame previously named "ForEach site" , add a **RegEx Extract** Action (contained in Platform Actions under the Beta Folder).

Configure the Action by:

- Adding in the field **String to Extract from,&#x20;**&#x74;he variable `HTMLString`
- Adding to the field **Regex Expression&#x20;**`([_\.0-9a-zA-Z-]+@([\-0-9a-zA-Z]+\.)+[a-zA-Z]{2,6})`
- Adding to the field **Lists of results&#x20;**&#x61; new variable named `EmailList `by hitting the Add Variable button-> Create New Variable. Once the variable is created select it to this field.

| Variable Name | Variable Type | List Value | Set as |
| ------------- | ------------- | ---------- | ------ |
| `EmailList`   | string        | ✔          | output |




![](/images/docs/wdcopjzdVi_jVQ8bZPeYq_image.png)



**Step 11. &#x20;**&#x49;nside the ForEach frame previously named "ForEach site" , add a **Concatenate Lists** Action (contained in Platform Actions under the Beta Folder).

Configure the Action by:


- Adding in the field **First List,&#x20;**&#x61; new variable named `MasterEmailList `by hitting the Add Variable button-> Create New Variable. Once the variable is created select it to this field.

| Variable Name     | Variable Type | Default Value | List Value | Set as |
| ----------------- | ------------- | ------------- | ---------- | ------ |
| `MasterEmailList` | string        | \[]           | ✔          | output |

- Adding in the field **Second List,&#x20;**&#x74;he variable `EmailList`
- Adding in the field **Concatenated List,&#x20;**&#x74;he variable `MasterEmailList`

![Concatenate Lists](/images/docs/WlJeQ8oyZE9jk2JN1WyAT_image.png)

**Step 12. &#x20;**&#x4F;utside the ForEach, on the Canvas, add a **DeDuplicate&#x20;**&#x41;ction (contained in Platform Actions under the Beta Folder).



![](/images/docs/BOYObn-qY4LdEHXOnhLEI_image.png)

Configure the Action by (select Action):


- Adding in the field **List to process,&#x20;**&#x74;he variable `MasterEmailList`
- Adding to the field **Duplicate Identification Json configuration**, `{"MatchType": "Exact"}`
- Adding to the field **Deduplicated list,&#x20;**&#x74;he variable `MasterEmailList`

![](/images/docs/7C4d1jSJoxeuhDsJMfvBX_image.png)



**Step 13.&#x20;**&#x41;dd a second **ForEach** Action on Canvas.

![](/images/docs/-ji7R0ZbuiiDKkRN5ZDro_image.png)

Configure the Action by (select Action):

- Configure field **For Each Item**, by typing `Iterator2` (this will automatically add a variable of type string to named `Iterator2`to the field.
- Configure field **In List&#x20;**&#x62;y pressing add variable button and adding the variable `MasterEmailList`
- Configure field **Action timeout**, by typing 600 (this impies that after 600 seconds, the process will timeout)



![](/images/docs/_kmM9qoI4llh5nF2Iy0wT_image.png)

**Step 14.&#x20;**&#x49;nside the second ForEach frame, add an **Add operation&#x20;**&#x41;ction.

Configure the Action by (select Action):

- Adding in the field **First Number**, a new variable named `Increment `by hitting the Add Variable button-> Create New Variable. Once the variable is created select it to this field.

| Variable Name | Variable Type | Default Value | Single Value |
| ------------- | ------------- | ------------- | ------------ |
| `Increment`   | integer       | 0             | ✔            |

- Adding in the field **Second Number,** the value: `1`
- Adding in the field **Result,&#x20;**&#x74;he variable `Increment`

``

![Add Operation](/images/docs/bfdvw7MIZW7nHiQvBCcSh_image.png)

**Step 14.&#x20;**&#x49;nside the second ForEach frame, add an **Call API&#x20;**&#x41;ction and in the side panel set the Node name to `SmartSheet`

**Step 15.** In Select API configuration select `SmartSheet`

**Step 16.&#x20;**&#x50;ress the **Configure Request** button and add these values:

Verb: `POST`
Endpoint: `/2.0/sheets/{Your Sheet ID}/rows`

- \&#123;Your Sheet ID&#125; was obtained in **Postman Step1**.



![](/images/docs/vog4Zw5RWIWy6qUbGCeWc_image.png)

Body:

```json
[
   {
      "toTop":true,
      "cells":[
         {
            "columnId": your column id,
            "value": <%Increment%>
         },
         {
            "columnId":your column id,
            "value":"<%Iterator2%>"
           
         }
      ]
   }
]
```

<Callout type="info">
Your column id is obtained in Postman **Step 2.&#x20;**(posting below a screenshot of how the Body of the Call API looks in PROCESIO



As observed besides the column Id(s), the body contains two variables: `Increment` & `Iterator2`

Additional hit, for inserting in the body of a Call API Action, one ca press INSERT key
</Callout>



![](/images/docs/rjH11AOvus4Uufs5IV0Xz_image.png)

**Step 17.&#x20;**&#x4D;ake sure that all your actions are connected in a sequence. Save the Process, Validate & Run.



![](/images/docs/nxzv2MFOGioeuashtp4MQ_image.png)

**Step 18.&#x20;**&#x49;n SmartSheet, check for new e-mail address entries.
